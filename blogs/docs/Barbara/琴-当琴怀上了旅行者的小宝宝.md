# 当琴怀上了旅行者的小宝宝

统计信息：字数6557  阅读14分钟  短篇小说

推荐指数：⭐⭐

 “事到如今，你还想跑？”

 旅行者刚想逃离琴的办公室，但是却被风压剑吸了回来。事情有些难以解释，看着琴现在的表情，要是不给出一个完美的答案怕不是下一刻就要被打飞出去。

 “那个，我怎么想跑呢？我本来想买点补品来着...”旅行者回头看向琴，虽说脸上的表情没有多大变化，但是在眉宇之间依旧能看出一丝恼怒。

  起因十分的复杂，中间的过程也不再赘述，若是按照普遍理性来解释那么就是琴怀孕了。当然现在这事情没人知道，凯亚不知道，丽莎不知道，甚至诺艾尔也不知道，只有旅行者和琴知道。

 孩子是谁的自然不用说，毕竟只有旅行者一个人晚上没事往琴的卧室里走，除此之外也没有别人，也就是丽莎没事情来看望琴一眼，当然这个时候一般旅行者都会藏进衣柜里。

 琴的衣柜可是旅行者觉得最奇妙的地方，一开始旅行者本以为琴是那种十分正经的人，穿着也应该十分的单一，但是没想到里面竟然有十分少女心的装扮，乍一看还以为是她妹妹芭芭拉穿的，

 其中最吸引人的莫过于蒲公英图案的胖次，实在是符合她的风范，又夹杂着一些少女心。旅行者还特意凑过去闻了闻，当然和她身上的香味是一模一样的。

 次数多了自然可能性也就越大，就像是向着河中投掷一千次石子，总有一次会砸到鱼儿。或许是哪次的安全措施没有做到位，事情就那么发生了。

 但若说是逃的话....旅行者这几天本来想去璃月散散心情，没成想被琴发现了，所以便有了现在的一幕。琴并不忍心伤害，她收回了剑，然后一泄气似的坐在了座位上，苦恼着思考着。

 “大团长回来要怎么解释....竟然出了这种事情...”

 “话说这是私人的事情，他没有那么大的管辖范围吧？”旅行者这时候在旁边说道，琴看了一眼旅行者，然后叹了一口气。

 “我是说最近的工作，我怕自己完成不了。”

 “那好办，我可以来帮你啊。”旅行者拍了拍胸脯自信的说道，可是他根本不知道琴的工作量有多少，虽然晚上的时候和她在一起，但是却很少和她一起办公。

 主要是去完成冒险家协会的委托去了，可能还回去璃月闲逛一番，当然，摸鱼也是有可能的。

 “你来帮我？有经验么？”

 “当然，我可是编了全璃月宵灯的男人。”旅行者笑了笑，随后坐在了一旁的桌子上。“快点开始工作吧，就算之后不能工作了我全权代理也可以，当代理代理团长。”

 “你可想的真好。”琴淡淡的说道，随后扔给他一摞子文件。“先帮我把这些处理完吧，不懂的可以问我。”

 “明白！”旅行者接过了文件，开始工作起来。

 时间一分一秒的过去，虽说中间也很多不知道的地方，但是在琴的帮助下大体上也明白了。中间两人有时候说笑着，有时候旅行者调情起来，但是琴的回应依旧是有些的....正经...

 不知过了多久，旅行者手里的文件终于全部处理完毕了，他伸了个懒腰，以为事情全部结束的他想要离开这里，带着琴去外面好好呼吸一下新鲜空气，但是刚站起身的那一刻，琴又拿过来一摞文件。

 “很好，热身结束了，接下来是这些。”

 这一摞子文件足足比旅行者还高，若是处理恐怕第二天他都处理不完。

 “啊？这也太多了吧？” 旅行者带着一丝不可思议的口吻惊呼道。

 “这只是基本中的基本。”琴坐回了自己的座位回答道，她手中的工作也有很多，即使是分给了旅行者一些工作，但是手头已经忙不过来。

 要是风花节有感动蒙德十大人物评选，那么以琴的敬业程度那么绝对高居榜首，其他人跟都跟不上来。颁奖典礼上全蒙德人聆听着琴团长的敬业光荣事迹，绝对听得老泪纵横，颁奖的时候可莉都要亲自送花，然后顺道被琴关禁闭。

 但是琴终究还是个少女，若非这样也必不会和旅行者在一起，在她的心理甚至旅行者比工作还要重要一些。

 “你累了么？累了就回去休息吧，我工作结束之后自然就会回去。”她叹了口气摇摇头，可是旅行者还是坐在原位，费劲的看着文件上的内容。

 “没事，我就呆在这里陪你。”

 琴不免嘴角露出一丝微笑，也许在不经意间她也可以享受这份小幸福，在间隙中不经意流露出来的幸福，就像是意外轻抚与脸上的蒲公英一样。

 又不知过去了过久，只知道工作终于结束了之后，蒙德的天色也黑了下来，虽说路上还有着行人，但大多数是巡逻的士兵，也有着一个不干正事的吟游诗人的身影。

 酒馆也没有多少人了，有几个喝醉的人躺在门口，骑士团的人正在费力的挪动他们，想要送他们回自己的家，可是那些人一动不动的躺在地上，像死了一般。

 “你的工作好累啊。”旅行者这一天下来觉得腰酸背痛，但是他却不急于休息，而是调情似的说道。“老婆，要不我们晚上再...”

 “不行，这个要求有些过分了。”未等说完琴便一口回绝了，旅行者要做什么她心里清楚，自己这个状况于情于理都不可能再继续了。

 旅行者也明白，他刚才那一句只是小小的开了个玩笑，当然还有更重要的事情要考虑。

 首先就是琴的身体问题，等到琴肚子大了以后肯定就没法工作了，到时候都要交给自己，但是交给自己要谁来照顾她呢？丽莎？

 其次就是可莉的问题，万一她顽皮起来伤到了琴，那说不好孩子就会...旅行者还是知道的，这个时候的女人是最脆弱的，只能用心呵护起来，就像对待细嫩的花朵一样。

 当然，还有一个最重要的问题...

 “对了，我们的孩子以后叫什么？”

 “.....”琴听到这个问题陷入了沉思，这个问题恐怕...

 要让他们思考好一阵子了。
