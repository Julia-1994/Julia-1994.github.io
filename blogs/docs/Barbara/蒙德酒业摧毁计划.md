# 蒙德酒业摧毁计划

​    对于习惯了自由散漫的蒙德人来说，每个周六和周日是绝对雷打不动的休息日——莫说是普通的工人和商贩，连西风骑士团都有着“拒绝内卷、合理摸鱼”的八字准则；然而，对于芭芭拉来说，这两个休息日却成为了她最忙碌的日子。

​    一方面是自己的身份特殊——西风大教堂的祭礼牧师，负责每周的礼拜主持、圣经朗诵和一些乱七八糟的教会事务，基本上每个礼拜日都要沉浸在精确到秒的工作中；另一方面，说起来还要怪到迪奥娜头上：这位猫耳调酒师近日推出新品的速度越来越快，不甘寂寞的酒鬼们为了光明正大地酗酒，把原本一年一度的“蒙德蒲公英酒节”，擅自调整成了一个月一次、最后甚至改成了每周六的固定节目；作为蒙德偶像，芭芭拉也不得不频繁出席于这种宴会场合，以满足蒙德群众日益激增的娱乐需求。

​    昨天迪奥娜推出的一款加了绝云椒椒和史莱姆凝液的清酒，彻底让酒鬼们陷入了疯狂；可怜的芭芭拉在众人的起哄声中，被迫在台上一直献唱到了凌晨三点多，最后还是琴团长亲自出面驱散了人群，才给这场闹剧画上了休止符。不过，虽说还是骑士团出面解决了问题和制造问题的人，但从团长到骑士，但凡是品尝过这款新酒的人，无一不对这款味道有些独特的清酒大加赞赏。

​    迪奥娜的“摧毁蒙德酒业”计划，毫无例外地又失败了……


    一场盛大的宴会过后，芭芭拉成为了唯一的受害者——仅仅睡了不到四个小时，芭芭拉不得不起了个大早，开始准备礼拜日的教会日程。

​    这位敬业的祭礼牧师，拖着疲倦的身体换好了自己洁白的修女服，费力地维持着身体平衡一路小跑，好不容易赶在教徒之前来到了西风大教堂。她在洗手间用冰水再一次让自己清醒过来，又在镜子前照了又照，确保自己依然以一个阳光、清爽、纯洁的正面形象示众，这才稍稍安下心来，敲了敲厕所单间的隔门。

​    里面传来了两声敲门声，示意里面有人，芭芭拉也没太在意——蒙德大教堂的洗手间设计得极其不合理，偌大一个洗手间装了四个水池，却只有两个隔间，其中一个前两天还被某个“很烫的红色女孩子”炸了个粉碎。若是在平时，教堂本就鲜少有人进出，倒也能勉强维持周转；然而，一到像礼拜这样的大日子，洗手间就人满为患，面对着少得可怜的坑位，女教徒们只得一边向巴巴托斯祈祷、一边苦苦忍耐自己的腹中之物，甚至还有过几次在厕所里失禁的记录。

​    对于芭芭拉来说，这种事情倒不经常发生——芭芭拉本身拥有水元素的力量，喝下去的水有相当一部分可以转换为身体的元素力，相比普通人来说自然不太需要小便；同时，习惯吃辣的她也有不错的肠胃，很少会因为刺激性的食物而拉肚子，也在很大程度上避免了大部分的腹泻尴尬场景。眼下自己虽然没有什么尿意，但毕竟一会儿要参加的是整个蒙德城最重要的教会礼拜祷告，芭芭拉还是习惯性地在祷告前去方便一下，将内急的危机概率降低至零。

​    等了一会儿，里面的人还没有出来，芭芭拉无聊地翻出了随身携带的圣经，顺便复习一下今天要讲的祷告词。

​    忽然，厕所的大门被猛地推开了，一个面色惨白的修女急匆匆地冲了进来。

​    “咦？罗莎莉亚小姐？”芭芭拉认出了眼前的这位修女。“早上好啊——你今天居然来上班了？”

​    “罗莎莉亚，你还好吗？”芭芭拉关系地问道。“你的脸色……看起来不太对？”

​    罗莎莉亚困难地点了点头，伴随着一阵轻微的肠鸣声，她猛地收紧了自己的屁股，手也往里捂得更严实了。“芭芭拉，我……”

​    “罗莎莉亚，你还好吗？”

​    罗莎莉亚大口喘着粗气，确保自己的下半身暂且无忧后，这才缓缓地开了口。“没事，大概是昨天晚上喝的酒……我已经好多了……”

​    罗莎莉亚的目光躲躲闪闪，整个脸红成了一个熟透的番茄，连声音都变得细若蚊呐。这是芭芭拉第一次见到罗莎莉亚如此害羞的样子，连忙拍了拍她的肩膀以示安慰，表示自己不会嫌弃。这位平日里高冷的修女，如今却像一个犯了错的小孩子一样，大气都不敢喘一下，小心翼翼地洗完了手，顺便将被自己弄脏的内裤偷偷丢进了垃圾桶，又将裙摆努力地向下拉了拉，以遮盖住自己撕裂的渔网袜和真空的下体，像一条夹着尾巴的丧家犬一般逃出了厕所。

​    怪不得罗莎莉亚看起来这么委屈……
