# 当神里绫华想为旅行者暖床

统计信息：字数5541  阅读12分钟  短篇小说


月光透过树叶的缝隙照射在地上，风轻轻地吹动着树叶，发出“莎莎”的声音。而月斑也随风而变化。

我行走在这寂静的森林里，前往和绫华的约会之地。

没过多久，我便听到了孱孱水流声。我拨开前面的草丛，然后发现溪水之中站着一位少女。

面容晶莹，肤如凝脂的少女在月光的照耀下如同散发着一股寒光，仿佛一个冰山美人。此时的月光就好比灯光般，周围的花草树木就是她的观众，而溪水之地就是她的舞台。

少女手持着一把扇子，开始了她的舞蹈。少女的舞台虽然有些不自然和不熟练，但是她依然保持着所应有的高雅和气质。

我被她那婀娜多姿的舞姿而着迷，在我的眼中，周围的一切事物都不存在，只有她一人。就好像是她只为我一人而舞蹈……

不知道过了多久，舞蹈结束了。而后少女收起扇子，脸有些微微透红，双手很不自然地在卷着自己的衣角。

“你、你来了……”

少女早已经发现了我，只是没有说破，而是直到她把舞蹈跳完。

“嗯。你的舞蹈很美哦。”我笑着走了过去。

“谢…谢谢夸奖……”

然后我们找了溪流边的一个地方，坐在石头上，而我们将脚泡在溪流里。

“这样和你一起看着月光，也不赖。”神里微微仰着身，双手往后撑着地面，头望着星空。

我扭头望着神里，精致白嫩的脸庞没有一点瑕疵，晚风轻抚，撩动着她的发丝，同时也在撩动着我的心弦。

此时神里注意到我正在注视着她，她的脸虽红，但是她没有躲开，而是鼓起勇气直视着我的眼睛，娇羞地说道：“我、我…好、好看吗……”

看着神里这副娇艳欲滴的模样，我的内心直挠痒痒，哪里忍受得住——

我伸手揽住她的细腰，将她一把拥进怀里。

“啊！”神里被我的这个突然而强势的举动惊到了，显然她还没做好准备。此时的她满脸通红，已经红到了耳根。她为了掩饰她的尴尬，便将头深埋在我的怀里。

我看见水中她的玉足在不停地拨弄着溪水，显然她对我这个举动并没有感到生气，反而是很开心。

想到这，我也是没有顾忌了，闻着神里身上散发的香气，笑道：“你一直都很好看。我游历诸国，见过的美人无数，但是能找得到和你相比的，恐怕是没有了。”

虽然知道这是我的甜言蜜语，但是神里还是会心一笑。

可就算如此，神里还是带着质问的语气询问着我：“你是不是也是这样骗其他女孩的？”

“哎？！”此时我冒着冷汗，“我…我不是很懂你的意思……”

而这时，神里皱着眉头，用力推开了我，从我怀里挣扎出来，嘟着小嘴，似乎对我的回答有些生气。

“你也是这样敷衍其他女孩子的吧。”

随后神里盯着我的眼睛说：“蒙德的芭芭拉、诺艾尔和优菈……璃月的刻晴、甘雨和凝光……还有霄宫、八重神子……甚至是雷电将军！”

神里为我一一列举了和我所接触过并且有交集的女孩。

“虽然稻妻现在锁国了，但是对于外界的一些消息我还是知道的。”

我还真是没想到，这位神里家的大小姐居然能把我的经历查的一清二楚……

神里一直盯着我的眼睛，似乎期待着我能给她一个让她满意的答复。而开始时我的眼神闪躲，并不敢面对她。

不过当我看到她那个期待和恳求的眼神，我心软了。而随之的是，我也下定决心面对着她，和她的眼神互视……

“啾。”

我俯身吻向了那娇艳动人的红唇。

被吻住的时候，神里还是愣住了。等她反应过来，我才缓缓地离开了她的嘴唇。

“你、你…你这是……”神里感觉到自己的心跳不断加快，身体有一种酥酥麻麻的感觉，就好像全身被电了一样，有些使不上力气。

“你不是想要我的答复吗？”我温柔地抱住神里，在她的耳旁轻声说道，“这就是我的答复，我的态度。”

神里的内心一暖，听到我的话后，她的嘴角微微扬起一个弧度。

“时、时间也不早了，我该回去休息了。”神里不舍地离开了我的怀抱，她的脸上依旧挂着两道醉人的红晕。

她此次出来是偷跑出来的，要是被托马发现那可就麻烦了。

“别走！”我拉住她纤细的手，然后走到她的身边，一把揽住她的蛮腰。

“夜还很长，我们再聊会儿吧。何况今晚的月色是那么迷人，如果有佳人陪伴在我身边的话，那么我会很开心的。”

而此时神里涨红了脸，我不知道她是生气还是害羞，然后她说了一句让我失魂的话，我瞬间就明白了——

“笨蛋，别让我说明白啊……”

神里的粉拳轻捶着我的胸口，然后借我出神的那一刻离开了我的怀抱。

没过多久，我回过神来，随之的而来便是欣喜，便对神里说：“这里环境这么好，在这也可以。”

“哼，我、我才没你这么心大，而且…我不喜欢在外面……”神里停下脚步，跺跺脚，“你、你还想不想…我为你暖床了……”

而随后我大笑几声，走过去以公主抱的方式将神里抱起，随后召唤出尘歌壶，向我的家走去。

一夜春宵值千金……
