# 【空X神里绫华 同人甜文】逐月佳节上篇：食与山河

嗯，不管你们信不信，昨天层岩刚出来我就连着肝了十一个小时，实在是没力气更新了，今天补上哈！

还是老样子，上篇传送门


从琥牢山回到旅店之后，已经是午夜了，“绫华，我们回到旅店了。”空想试着叫醒怀里的绫华，却发现她已经在自己的怀里睡着了，脸上的表情写满了“满足”二字，而且睡得很熟，空在利用风之翼飞着带绫华回旅店的时候都没醒来过。

 

他也没再想着叫醒绫华，非常温柔的将她放上了床，盖好了被子，便自己一个人悄悄的洗漱去了——本来他是有吃宵夜的习惯的，只不过现在这个钟点，璃月港内的商铺几乎都关完了。

 

只不过就是时间允许，空现在也不想出门了——他和绫华在琥牢山大战了一场，现在空的腰甚至还有些疼，“嘶——果然这件事还是得在家里...还是睡觉去吧......”想到这，空现在只想回床上休息了。

 

为了不吵到正在熟睡的绫华，空钻进被窝的动作非常的轻，但在准备盖上被子睡下去的时候就被绫华一把抱住了，“不...不许离开我......”说话的时候眼睛没睁开，也不知道是在说梦话还是故意的，但空也没管，轻轻地抱住了绫华，就这么睡了一晚上。

 

隔天早上一醒来，绫华看着身边抱紧自己的空，瞬间就回忆起了昨晚在琥牢山上和空所做的一切，然后打了个喷嚏，吸了一下鼻子，还打了个寒战。

 

“果然还是着凉了吗...啊嚏！噫...”“呃...绫华......”

 

“嗯？......啊嚏！”又打了个喷嚏，这回是让空彻底清醒了，“呃...果然还是着凉了吗......”

 

“呃...没事啦，只是有点...啊嚏！”

 

“你先去洗漱一下吧，以防万一，还是带你去不卜庐给白大夫看一下吧。”

 

“嗯...好吧，那你也得起来了喔。”

 

“我知道，你先去洗漱整理一下吧。”

 

两人起床之后，洗漱整理啥的到也没花多少时间，只不过呢，出门前空又给绫华加了一件薄衣物，“本来昨晚就有些着凉了，要是再着凉那可不好了。”

 

“还不是空你要在那个地方做......”

 

“喂，还不是你说可以我才继续的。”

 

“我！......”“好啦好啦，不逗你了。走吧，带你去喝早茶，先暖暖身子，完事再带你去不卜庐给白大夫看看，省的到时候一个普通的风寒成了什么严重的大病，我会很心疼的呀！”

 

“嗯...”

 




在璃月，早茶是个挺有趣的一个传统：生活在璃月的老百姓们，如果早晨有时间的话，便会在万民堂这边买上一桌早点，邀上三五个好友一起坐在一桌，一边喝茶，一边吃早点，一边聊聊那些生活琐事，当然啦，那些喝早茶的大多都是老一辈的人，很少在早晨的万民堂见到年轻人的身影，绝大多数都是些老大爷，三五个围坐一桌，时不时张嘴聊聊生活琐事，比如隔壁二大爷养的鸟天天乱飞乱叫，对门三大爷养的狗成天乱跑啥的......总之就是一大堆的生活琐事，啥都有。（两广人嘛，早茶这东西算是家常便饭了，诶嘿！虽然早茶在广东比较普遍？）

 

“绫华，你们稻妻那边有喝早茶的习惯不？”空刚好联想起了稻妻那边似乎没有喝早茶的习惯。

 

“没有诶，我们稻妻那边吃早餐还有分官位的，像我们神里家，早上都是梅干煮豆子，配上烧酱，两菜一汤，挺简单的。诶？那璃月的早茶是什么呀？”（这里稻妻的早餐参考了日本江户时代幕府将军的早餐）

 

“璃月的早茶嘛，我觉得其实就是丰盛一些的早餐，就你可以叫上三五个知己好友一起围在一桌，边吃早点边聊家常，当然啦，在璃月这边，早茶很少有年轻人在喝的啦。”“诶？这不是鼎鼎大名的旅行者嘛！能够在这个时候这个地方见到你属实荣幸啊。”空刚说完话，行秋和重云便刚好看见了走在大街上的空和绫华，刚好就凑了上去打了声招呼。

 

“呀，是行秋和重云啊，这么早是要去喝早茶吗？”

 

“正是。既然见到了你们，那不如，我们一起？算我请客，就无需劳烦二位破费了。”

 

“二少爷这么说我可就不困了！绫华，我们走吧。”

 

“嗯。”

 

来到了万民堂，第一眼看见的不是卯师傅，而是他的女儿香菱，“呀，旅行者，你们来啦！今天是来喝早茶的吗？”

 

“那个...这位小姐，不知您有什么忌口没有？”

 

“哦，辣的东西她吃不来，她喜欢清淡些的。哦还有，动物脂肪和内脏她吃不惯，所以，麻烦了。”

 

“那就好。一份经典早茶套餐，补上两壶上等好茶，谢谢。”

 

“嗯，好的，各位请稍等片刻，早茶马上就来！”

 

行秋和空他们四人找了个位置坐了下来，这个时候行秋才开口问了一句：“旅行者，试问这位小姐是......”

 

“稻妻神里家长女神里绫华，请多指教。不知二位该如何称呼。”

 

“我叫行秋，区区飞云商会二少（xiao）爷（jie）罢了，这位是方士重云，我的朋友。”“我是方士重云，请多指教。”

 

“请多指教。”



“稻妻神里家我有所耳闻，没记错的话，是稻妻三大奉行之一的社奉行，对吧？前些日子对稻妻神里家大小姐来璃月这事有所耳闻，今日相见，神里家大小姐可谓是名不虚传啊。”



“啊，行秋少爷过奖了，我只不过是...”“喂，行秋，你可注意点，她现在是我的人了。”



“空！”“哈哈！放心吧旅行者，没有跟你抢的意思！”

 




“你们的早茶来了，请慢用！”“哟，旅行者和神里小姐也在这啊，真巧啊。”香菱刚将行秋他们点的早茶套餐，刻晴，凝光还有甘雨后脚也来到了万民堂这边喝早茶了。

 

“刻晴，今天这么悠闲，能在这个时候来万名堂喝早茶？”

 

“是啊，凝光小姐将近段时间的工作全部推掉了，让我们出来好好休息几天。”

 

“那，既然来都来了，不如...一起？”

 

“嗯，可以。”

 

“那...香菱，再来一份经典早茶套餐。”

 

“好的！”

 

随着凝光刻晴和甘雨的入座，这一桌的人便都开始聊了起来。

 

首先是刻晴开了个头：“旅行者，近段时间我们璃月的传统节日，逐月节就要开始了，不知道接下来你是怎么打算的？”

 

“我嘛，会和绫华在这里待上一段时间，毕竟我们也是稻妻方面派来和璃月进行商业合作的负责人，要想离开璃月的话估计得等上一段时间。”

 

“既然这样，还希望旅行者你能够在逐月节这段时间带上你的妻子出来璃月街道上好好逛逛，正好让神里小姐感受一下异国节日的氛围。”

 

“嗯，我也是这么打算的，最开始我们也确实是为了逐月节而来，只是刚好璃月方面给到稻妻大使这么一封信，我们也就顺手接下了这个工作了。”

他们一行人开始喝早茶的时间大概是早上八点，现在人多了起来，空和绫华他们也聚在一起聊了有好些时候了，十点钟这样子，大伙们也各自散会了。

 

时间也是过得很快，三天之后便到了一年一度的璃月传统节日——逐月节。

 

“走吧绫华，我们上街看看！”

 

“好！”

 

整理了衣着，走出了旅店上了街，全璃月港的氛围跟三天前的差别大了不少——虽说没有海灯节那样张灯结彩，但街上的行人却是比往常的时候多了许多，整个璃月港的热闹程度可不比海灯节的时候差。

 

“嗯，今年的逐月节还是一如既往的热闹呢。对了，我们去万民堂走一圈吧，据说今年的逐月节，七星为了迎合今年逐月节‘食与山河’的主题，特意举办一场厨王争霸赛，万民堂老板的女儿香菱前几天就叫我帮她试菜了，一起去吧，没准你们稻妻那边在食品方面的特色能够被她借鉴呢。”

 

“走吧！”

 

走在了大街上，看着络绎不绝的行人，空似乎想起了什么：“对了绫华，你们稻妻那边有什么传统节日吗？”

 

“传统节日...我也没听说过，可能是因为之前稻妻的眼狩令和锁国令还没解除的关系吧，当时整个稻妻的气氛都非常的压抑，人们都没什么心思去过节日，祭典似乎已经是当时我们稻妻最大规格的娱乐节目了。”

 

当时的稻妻，气氛确实是非常的压抑，在锁国令和眼狩令的双重压迫之下，整个稻妻城死气沉沉的，让别人来看甚至不会以为这是整个国家最繁华的地方，唯一能让这死气沉沉的稻妻活过来的，怕是只有甘金岛上的祭典了。

 

“倒也有道理。唉，希望将军大人能在眼狩令和锁国令解除之后想开一点吧。”稻妻那边，毕竟眼狩令和锁国令的废除都是空一手参与进来的，他知道稻妻的雷电将军——或者说是影，在废除了眼狩令和锁国令之后还有很长一段路要走，略微的有些担忧稻妻会在影的直接领导下向着什么样的方向发展。

 

“不过，看现在稻妻这个样子，事情似乎也在向着好的方向发展呢！”绫华身为稻妻三大奉行之中掌管民生的社奉行，对于眼狩令和锁国令解除之后稻妻城内百姓生活的变化非常清楚，她也知道空这么说是在为了稻妻将来的发展感到担忧，便出言安慰道。

 

“嗯，确实是这样。”绫华这么说也是有道理的，毕竟，在空时隔两年再次踏上稻妻国土之后，他能很明显的感受到整个稻妻城的变化，绫华所说的这句话，倒也不用过多的证明就可以确认其真实性。

 

“呀，是旅行者和神里小姐，我们又见面啦！”两人交谈之中便正巧来到了万民堂，香菱刚好也在万民堂里做菜。

 

“是呀，又见面了。对了，你说你最近在为了厨王争霸赛做准备，叫我们过来试试菜来着？”

 

“嗯...比起试菜，我倒是更想请教一下你们的口味如何。”

 

“我们的口味？”

 

“换句话说，我想请你们帮我收集一下大家对于美食的想法，比如说...旅行者通常喜欢什么口味的菜？是咸一些，辣一些，还是清淡一些呢？神里小姐又是怎么想的呢？”

 

这个问题从香菱口中说出来那是很有深度的，空倒也不敢随便回答一句就应付过去了，便左手叉腰，右手放在了下巴上做出了思考的动作。

 

“嗯...我之前一直是在外旅游，对于菜品口味的要求真的不是很高...真要说的话，我到还是相对热衷于清淡一些的菜品。绫华你呢？”只不过吧，空之前确实是一直过着餐风露宿的生活的，对于菜品的要求到只有一点——能吃就行。香菱的这个问题在他这边...说实话，挺难回答的上来。



这个回答或许是空能想到的最好的回答了。

 

“在我们稻妻，菜品的口味都比较的清淡，我也挺喜欢清淡的菜品。”绫华的回答就非常自然，毕竟她不像空一样成天餐风露宿，这每日三餐都是准时来的，而且讲究也多，真要说那一时半会也说不太来。

 

“清淡点的呀...诶对了，神里小姐，你们稻妻有什么不同于璃月这边的菜品不？”

 

“这个嘛...甜品的话，我挺喜欢绯樱饼的。”

 

“绯樱饼...啊，这个我知道！是不是用晒干的绯樱绣球做的？”

 

“对呀，稻妻人经常将绯樱饼作为饭后甜点。对了，要说饭后甜点的话，三彩团子也不错，宵宫来我家做客的时候就经常带给我吃呢。”

 

“绯樱饼，三彩团子，当饭后甜点...嗯，我记下了。走吧，我们去找其他人问问。”

 

“诶？锅巴去哪了？”这个时候空才想起来之前香菱身边一起跟着的锅巴不知道去哪了。

 

“锅巴也回来了，现在在门口翻跟斗呢。”

 

“喽？”听到香菱叫到了锅巴，锅巴就屁颠屁颠的从万民堂门口跑了进来叫了一声。

 




带上了锅巴，三人在谈笑之间便来到了往生堂，就听到了一阵非常熟悉的声音：“太阳出来我晒太阳，月亮出来我晒月亮咯！喂！七七，别跑啊！”

 

“七七...不要...被抓...”

 

“得，又是熟悉的场景。胡桃，别老缠着七七了。”空算是看惯了胡桃在往生堂附近追逐七七的画面了，和绫华香菱两人站在不远处看着胡桃说道。

 

胡桃听见了空的声音，也迅速停了下来，展示了一手她经典的单脚刹车，站在了原地转头看向了空，刚好这个时候空也一把抱起了正巧撞上他的七七，并顺势把她放在了一边。

 

“哟，找本堂主有何贵干啊？”

 

“你好啊胡桃，我们来是想像你请教一些问题的。”

 

“哦？请教我一些问题？嗯...这几天是逐月节，总不可能是那个方面的事情吧？”

 

“哎呀不是啦，我问你，你对菜品的口味有什么讲究吗？”

 

这个时候胡桃有些的愣，那棕褐色的梅花瞳机械般的眨了两下，才缓过神来：“对于菜品的口味...你要是问这个我一时半会还真答不上来呢。”

 

“诶？”空倒是挺诧异的，居然还有胡桃回答不上来的问题。

 

“没事的胡桃，简单说说就行。”

 

香菱和胡桃交谈之际，空和绫华身旁的七七现在和一只刚巧飞到她面前的小团雀玩的正开心呢：“咦，团雀。七七想要...小团雀...”在那只团雀从她面前飞走之后，七七跟着团雀飞走的方向看了一遍，最后指了指绫华的肩头，“小团雀...”

 

“嗯？”绫华这才注意到有只小团雀停在了她的肩头之上，“七七喜欢小团雀吗？”

 

“七七...喜欢...小团雀...”七七甚至还轻轻拉了一下绫华的裙摆。

 

这个时候，那只小团雀并没有飞走，而是非常老实的在绫华的肩头上站了有好一段时间，甚至还和绫华的眼神对上了，一直到绫华将手指轻轻的伸到了小团雀面前，小团雀才“扑腾”一下飞到了绫华的手指上。

 

“呐，七七，手伸出来，你喜欢的小团雀要来咯。”绫华现在手指上停着一只小团雀，她知道七七是想抓这只团雀，动作便放的特别轻，生怕惊动了这只团雀。她俯下身子来，将手指上的团雀小心翼翼的放在了七七刚伸出来的双手上，“小心点，别弄伤他喔。”

 

“嗯，七七，会小心的。”

 

“看来绫华挺擅长和小孩子打交道的嘛（虽然七七是个活了三千年的小僵尸）。”空是看着绫华和七七交流的全过程的，就觉得绫华很擅长和小孩子交流，一个大胆的想法便从他口中说了出来：“看样子绫华以后也是个好妈妈呢。”

 

“诶？！当妈妈什么的...还太早了点吧......”被这么一说，绫华又脸红了。

 

“嗯...谁知道呢，没准很快，没准很慢呢。”

 

“嗯......”

 

也没过多久，香菱便和胡桃聊完了天，准备同空和绫华两人带着七七去不卜庐走一趟，正好顺路送七七回去。

# 【空X神里绫华 同人甜文】逐月佳节下篇：民以食为天

今天双更兄弟们！惊不惊喜意不意外？！

还是那句话，上篇传送门


一路上有说有笑的，唯有七七跟手中的小团雀玩的正欢，那只小团雀似乎也挺喜欢七七的样子，一直待在她手心上不走了。

 

就这儿一路来到了不卜庐，“那个，白术先生？”空看见不卜庐里面没人，便小声的叫了一下白术。

 

“哦？稀客啊，看来你们今天难得清闲了下来。”一个稚嫩的声音从不卜庐里面传来，随后出来的便是白术的身影，脖子上还有一条蛇缠在上面。

 

随后跟着的是成熟又带点温柔的男声：“诸位，好久不见了。”
