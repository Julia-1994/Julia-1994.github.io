# 社奉行家的绫华大小姐怀上了空的孩子

清晨的稻妻

伴着鸟语花香

躺在绫华旁边的空伸了伸懒腰

张开了眼

空转头望了望绫华

细心地盖上了被子

......

空起床后洗了把脸

坐在了窗边

“这么早？”

一旁的绫人走了过来

“嗯...不知道为什么”

“怎么样 习惯吗”

“还...可以 嗯...”

这时绫人坐在了空的旁边

“话说...你没做什么吧”

绫人一把凑在了空的面前 严肃地问

“没没没...不敢”

空慌张地说

“嗯？真的？”

绫人半信半疑的问了一下

“真...真的”

“那就行...你肚子饿了吗”

“有点了”

“那跟我来吧”

......


“嗝...”

空坐在椅子上打了个嗝

“怎么样 我的厨艺不错吧”

一旁的托马说

“不错 厨艺确实有所进步”

绫人说

“嘿嘿 谢谢家主大人的夸赞”

“那个...空 你去把绫华叫醒吧”

“好 我知道了”






空悄悄地走到绫华旁边

看了看绫华

熟睡的绫华脸上泛着淡淡红晕 嘴巴微微张开 

看到绫华熟睡的样子 空不忍心叫醒她

这时 绫华醒了过来

“唔...早上好 空”

绫华揉了揉眼睛

“嗯...早 起来啦”

“不嘛...抱抱我...”

绫华对空撒着娇

“好好好”

空坐了下去 一把抱起了绫华

“没想到往日高贵的绫华大小姐也会有这么可爱的一幕啊”

空在内心不禁感叹

绫华披散的长发落在肩上

梳妆台前 空正帮着绫华梳理着头发

......

“好啦 这样就可以了”

“哇 谢谢了”

“嘿嘿...不用谢我...”

“那个...空 我有个问题”

绫华突然严肃起来

“什...什么问题”

“就是...你还能在这暂住一晚吗...”

“这个...如果可以的话...”

“留下吧...”

“啊...好...”

绫华激动的跳起来 亲了空一下

“太棒了”

“绫...绫华...不用这么激动...”

......






晚上 绫华的房间内

空正帮绫华叠着衣服

“空 你能留下来实在是太好了”

“嗯...只不过是怕绫人他...”

“没事的 兄长他不会对你做什么的”

“但愿吧...”

......

“啊～”

绫华打了个哈欠

“困了吗 困了的话就睡觉吧”

“有...有点了”

“那就睡觉吧 我也要睡了”

“嗯 好”

......

“那个...绫华...我有个要求...”

空贴在绫华的耳边悄悄地说

“什么要求？...”

“就是...”

这时空抱住了绫华的腰

“空...你...”

空看到绫华涨红的脸后更兴奋了

“绫华...跟..跟我...”

话还没说完 绫华便打断了空

“我...我知道了...其实...我也想...”

这让空感到非常震惊

“绫...绫华...你...”

“没关系的...今晚...请随意使用我...”

说罢 绫华解开了衣服的扣子

脱下了上衣 趴在了空的胸前

“空...我还是第一次...请..请轻一点...”

空摸了摸绫华那涨红的脸蛋

“好...我知道了...”

......

“绫...绫华...没事吧...”

“最...最喜欢空了...”

趴在空怀里的绫华说

......
