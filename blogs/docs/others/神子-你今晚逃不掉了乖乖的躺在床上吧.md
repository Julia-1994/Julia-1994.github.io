# 神子：空...你今晚...逃不掉了...乖乖的躺在床上吧...（原神/桃文/同人）

微风吹过夏日的稻妻

远处的日落映射在一旁的纸门上

落下的樱花叠成一片片

空的怀里正抱着他和神子的孩子

“乖乖...睡觉觉”

这时神子走了出来

“空 你累了吧，我来抱吧”

“嗯 好，对了 今晚影他不回来了吗”

神子接过空的孩子 说

“她在天守阁有事要处理 今晚就不回来了”

“啊...这样吗”

“嗯...你肚子饿了吗”

“有点了...我来做吧”

“不...不用 今天我来吧”

神子脸上惊慌的表情并没有让空察觉到

“那好吧 我也好久没尝过神子的手艺了”

“那你把孩子哄睡了噢 我去做饭了”

“嗯！”

空接过神子怀里的孩子

“宝宝...睡觉觉啦...”

神子径直走向厨房

“空...你今晚...是属于我的了...”










晚饭过后

空坐在椅子上

“嗝...好饱啊 神子你的手艺果然没有退步呢”

“那当然 我是谁呢”

“也是哈...”

“对了空 你渴不渴呀”

“嗯...有点”

“那我给你倒点水吧”

“好好”

神子走向了厨房 一脸坏笑着

“这个...好像是倒这的吧”

说着 神子就给空的那杯倒下了药粉

“这个不知道喝了会怎样呢...我也来一点吧”

神子又给自己那杯倒了一点药粉

“这样就行了”










“啊...有点困了...”

空打了个哈欠 走向了卧室

“空？你在哪呢”

“我在卧室”

“正合我意呢...”

神子此时心里想着



此时卧室内

“来 这杯是你的”

“嗯 好”

空接过神子手里的水 一股脑喝了下去

“这水 味道怎么怪怪的”

“你想多了吧 有啥怪的”

神子也把自己的那份喝了下去

“好像是有点怪怪的 算了不管了”



突然 神子一头栽向了空的怀里

“神...神子...你怎么了！”

“空...你热不热啊...”

空看着脸上泛着红晕的神子

“没有啊 我一点都不...”

“奇怪...怎么会...”

“嘿嘿嘿...你也中招了吧...”

“神子..你..下什么了...”

“我只不过是...下了一点点...春药而已...”

“神..神子..你...”

话还没说完 空就睡了过去

“其实...我还下了一点安眠药...”

神子熟练的解开空的衣服

“让我好好品尝你吧...空”
