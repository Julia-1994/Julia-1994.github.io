# 【原神/丽莎/桃子】沙漏星象

（本篇无车，只有糖～时间大概在丽莎被表白后一个月左右～）


　　丽莎深吸一口气，解开发带。头发顺着她的天鹅颈垂下，层层的涟漪、熠熠生光，恰似一道淡金色的瀑布，微微打着卷。她略显羞涩地把头发快速挽起来，踌躇了一会儿，然后站定，拎起裙角优雅地转了一圈。

　　她眨了眨眼，仔细审视镜子里的自己，裁剪合身的紫色过膝裙，一双白玉长腿在红地毯上随意踢踏，发出咚咚的清响。烛光的映衬下，精致的妆容配上妆点在唇上一抹雍容的鲜红，丽莎仿佛年轻了好几岁。

　　“他说过他喜欢单马尾的……是吧？”丽莎自言自语道。不断变换姿势，欣赏着镜中的倩影。一向自信慵懒的丽莎小姐，今日看上去竟有些紧张。像是要和男友出去约会的少女，反复确认自己是否打扮得足够惊艳，但却一直犹豫不决。

　　“就这样吧，他会满意的。”丽莎对着镜子里的自己重重地点了一下头，试图自我催眠。随即，她戴上神之眼，裙子一摆，飞快地出门下楼，来到蒙德的大街上，眼睛里闪烁着星空的光芒。

　　正值六月，蒙德的空气愈加凉爽。这是个茂盛的季节，躁动的暖流在暗暗发酵，一如她对他的爱，炽烈但却绵长。如果非要用什么事物来形容的话，一杯清冽的蒲公英酒足矣。

　　初见丽莎的人，往往会对她形成「不愧是教令院天才毕业生」的印象。无论是荒野妖魔的禁忌知识，还是元素花药的处理方法，丽莎都能简明扼要地解释其中的原理，只要在炼金术上遇到难题，蒙德人一定回去问问无所不知的丽莎小姐。

　　那些春心萌动的小伙子眼中，丽莎是个成熟撩人的大姐姐。每次去借书时，她也都能感觉到他们的爱慕，时不时说几句暧昧不清的话撩得他们不能自已。以此为乐。

　　丽莎自以为对男人了如指掌，可当爱情真正来临的时候，她和情窦初开的少女并无二异，依旧是一无所知。每当对上他真诚的目光，伏在他温暖的怀抱……绕是镇静如丽莎，心脏也避免不了砰砰地跳。实践和理论，并不总是相同的。

　　直到现在她才明白，真正的两情相悦是不需要任何技巧来维持的。只需要两颗毫无保留的心，一个懂你的人，便可以让你在与他相处的时候如置天堂。

　　就这样漫步在蒙德的街头，十指纠缠在一起背在身后。颔首低眉，内心万千思绪。现在的丽莎，美得像没有刺的蔷薇。在街道上前行总会带起一阵清风。男人们无一不驻足观望。美若天仙，这是此刻他们内心仅有的词，只出现在小说诗集里的词。虽然谁也没见过天仙究竟美得如何，但看到丽莎，他们不约而同的有了一个共同的标准。

　　她美得发光，照亮了每一个角落，每一个人。

　　“他今天约我去酒馆……是为什么呢？既然不是什么特殊节日。难道……”丽莎晃晃脑子，试图驱散那些不正常的想法，全然没注意到迎面而来的少女。

　　“Duang～”“呀——”少女撞进丽莎的正义之中，发出了嘹亮的尖叫。丽莎急忙抬起头，芭芭拉正站在她的身前，小嘴张得大大的，一副惊讶神情。

　　“你是……丽莎姐姐？！”芭芭拉难以置信地看着她，牧师帽歪向一边。“你今晚的打扮……我第一眼真没认出来……”

　　“是吗？”丽莎优雅地问道，但掩饰不住双颊上那一抹绯红。

　　“嗯嗯！”芭芭拉头点得像打桩机，“你这样成熟美丽的大姐姐，要是也成为偶像的话，我绝对会全力应援、喊到嗓子哑掉！啊，这样会吓到别人的吧……”

　　“噗……”芭芭拉惊慌失措的模样，让丽莎忍俊不禁。随即，她低下身，摸了摸芭芭拉的脑袋，愉快地说道:“也不是不可以哦，如果哪天我不想当图书管理员了，可以考虑一下呢～”

　　“诶诶诶，真的？！”芭芭拉眨了下蓝色的大眼睛，“说实话，看到你的第一眼，我把你认成了琴团长……”

　　“有吗？”丽莎理了理头发，歪头问道。

　　“真的很像！”芭芭拉再一次点头，随即露出明知故问的表情，“话说……你是去找雷明顿哥哥的吧？”

　　“嗯。”丽莎微微将头转向一边，双眼隐藏在秀发的阴影之下，刚刚平复的心情再次翻涌起来。

　　“那我就不耽误你啦～”芭芭拉话还没说完，就飞快地跑了出去，像长了玫红色的翅膀。

　　“芭芭拉冲呀～”丽莎下意识地说喊了这句话，高举右手用力挥舞着，欢呼着，仿佛芭芭拉是第一个冲过终点线的百米运动员。随着目光上移，她的身影最终消失在了石阶拐角处。


　　傍晚的夕阳逐渐落幕，阿尔忒弥斯披上星星做的斗篷，一只手支着下巴优雅地躺在星空之上，慈母注视着提瓦特大陆。星光一如洒在蓝布上的碎银，蕴藏着少女不可告人的小秘密。

　　“他今晚会是什么打扮呢？”丽莎抬起头憧憬道，也许答案就在不远的酒馆。她加快脚步，转过街角，就是「天使的馈赠」了。

　　风变得急躁起来，她的心也难以按耐，可她还有些踌躇，想要转身走回去再整理一下自己的仪容。刚才风有些大，一定把头发吹乱了，要是让他看见，该怎么办……

　　终于，还是没停下脚步。急切的心战胜了羞涩，驱使着丽莎走完这咫尺天涯的路程——她看见了他，都是彼此魂牵梦萦的人。

　　一如既往的单片眼镜挂在他的鼻子上，纯黑色西式套装服帖地穿在身上，他看起来像个学者，稳重安静，让丽莎倾心不已。

　　我坐在露天的酒桌旁，放下手里摆弄着的黄铜望远镜。抬起头瞧见眼前的美人，一时间说不出话来，我只能呆呆地看着丽莎，下意识地动了一下喉结，甚至忘了给她倒饮料。

　　“抱歉，你今天……我没认出来……”我连忙打开那瓶冰钩钩果汁，丽莎不喜欢喝酒，我也一样。

　　“还不是因为你，我才把自己打扮成这个样子。”丽莎有些紧张的脸瞬间放松下来，闪跃着愉悦的火花。

　　“坐吧。”我拍了拍身旁的椅子，丽莎乖乖地坐下。屁股一挨到椅子，她立刻软了下来，黏在我身上不肯离开。丽莎今天没喷香水，淡淡的体香依然让我陶醉。

　　我和她都不喜欢在约会时挑面对面的座位，那样总会有一股莫名的空虚感萦绕在心头，仿佛下一秒我和她就会分开。唯有紧紧贴在一起的感觉才会安心。

　　“你还没告诉我今天为什么请我来呢～”一杯冰钩钩果汁下肚，丽莎感到好了点，开始端详起杯中的紫色液体，仿佛为她而生的颜色。

　　“和你在一起的第一个月，不记得了吗？”我微微一笑，把放到一边的黄铜望远镜又拿了过来，“上周，莫娜在酒馆里对我大放厥词，说今晚会有明显的星象变动。正好，我想趁这个机会约你出来……”

　　“和爱人依偎在一起，欣赏夜空中划过的淡蓝色浪漫火花？可是，亲爱的，这样老套的套路，可没法博得女孩子的欢心哦～”丽莎看着我，语气略显失望，但却将我抱得更紧了。

　　“无论是在须弥教令院还是西风骑士团的图书室，以前的那些时光里，你从没像现在这样和一个男孩子挨得这么近吧？——谢谢，”我接过查斯曼递来的糕点，继续说道，“这会是一场不一样的体验。”

　　“可别让我失望～”丽莎轻声细语道，“不过话说回来，莫娜最近怎么样了？虽然我不是「魔女会」的成员，但她可是一名优秀的魔导师呢。”

　　“她最近在蒙德的冒险家圈子里混得风生水起。我不得不说，冒险家协会真是一个神奇的地方，只要有梦想，有足够的能力和胆量，就可以用最快的方式为自己整得荣誉……”

　　“对你们这些喜欢到处跑的人来说这很简单，可我这样的懒人而言，冒险真是件令人苦恼的麻烦事。”丽莎把被压住的马尾辫抽了出来，发丝拂过脸颊，痒痒的。

　　“我刚来到蒙德的时候，无依无靠，若不是赛克琉斯收留了我，我可能还在荒郊野岭游荡呢。”

　　“所以，姐姐我的作用就那么卑微吗。”丽莎委屈巴巴地抬起头，眼中泛起点点泪光。不得不说，她楚楚可怜的模样足以让一大批男人为之疯狂，“你作为一名读者，我却给了你图书室的钥匙，赋予你24小时内自由出入的权利。在炼金术上得到了我的指导。而且，还可以和姐姐独享美好的下午茶时光……那可是让蒙德城无数小伙子都为之眼红的机会呢～”

　　“可是，你这么迷人的姑娘对我发出邀请，我怎么会拒绝呢？”我有些不要脸地说道，“到图书室读书的时候，我最惬意的时光，就是放下八百页的《绪论》看着你把两条长腿放在桌子上，一边喝茶一边晃动脚丫……那时候我只有一个念头，”

　　“什么念头？”丽莎不但没有生气，反而似笑非笑地看着我。我看到凯亚正在不远处的一桌上和霍夫曼喝酒，他回过头来对我不怀好意地笑了一下。

　　“去他的《魔导绪论》，还有该死的《高阶炼金术》，还是你更好看。”我诚实地说道，一副一本正经而不失温柔的表情。

　　“我早就猜到了，啾～”丽莎抿着嘴唇，在我的脸上留下风铃子味道的吻，“这种略显轻佻的话本来会惹我生气，但你这副认真的表情让我怎么也愤怒不起来，更多的是感动……”

　　“陷入热恋的女孩子，无论情人说什么，内心都会泛起一股柔情蜜意吧。”我抬起头望向天空，在略偏西北处的带状区域，那里的星星似乎比其他星星更加耀眼。

　　“这并不完全如此。爱情是最难以破解掌握的魔法，直到我还在门外徘徊呢。”丽莎摆了摆手，“在你之前，也有很多小可爱对我表白，想方设法接近我，其中不乏优秀之人。可我一点感觉都没有，无论他们把求爱仪式搞得再怎么隆重富有创意，都不及你对我的回眸一笑，或者是一句「晚安，亲爱的」。”

　　“这叫一见钟情，或者见色起意。”看着丽莎扑闪扑闪的大眼睛，不知为何，我有些想笑。

　　“不，这不一样。直到后来我才明白，你是能抵达我内心深处的人……当所有人都为我的美貌而倾倒的时候，只有你理解我……”丽莎喝光了杯里的果汁，再倒上一杯。

　　“我知道，”我轻轻抬起丽莎的下巴，凝视她迷茫的绿瞳——她和我拥有一双同样颜色的眼睛。随后，我叹了一口气，复杂地说道:

　　“你我都是被知识诅咒的人。”

　　亲眼目睹须弥雨林中呢喃狂言的学者与评议会中如有大智慧的学者之后，丽莎深深理解了渊海般的学识会给人烙下怎样的痕迹。代价如此沉重……到底要承担多少，才能从灵魂深处挖掘出这样的知识呢？

　　丽莎离开了须弥，从那以后，不再对任何事物展露出完全认真的态度。除了……对他的感情。

　　丽莎趴在桌上，只露出一只眼睛偷偷看着他。她还记得那晚他的话语，「如果知识需要代价，那就让我来支付吧」。第一次，自己寒风吹拂的脊背有了依靠，自己淡然的心在剧烈跳动。丽莎头一次想成为他的女人，不用每天和刁钻的读者和繁杂的书籍打交道，可以待在家里做自己喜欢的炼金实验有大把大把的时间可以自由支配……

　　真爱只需两情相悦，互相包容理解，再加上一点日复一日的新鲜感，便可以永葆青春。

　　“嘿，发什么呆呢？看天上！”我对丽莎轻声细语地说道。她从沉思中抬起头，脸颊微红，像极了受惊的小兔子。如果真能在她脑袋上安一对兔耳，未尝不是一个好主意。

　　“啊？啊！”丽莎呆呆地抬起头，迷茫地望向天空，“在哪儿？”

　　“顺着我手指的方向。”我一边用望远镜看着天空，一边用手指着刚才星光耀眼的区域。酒馆的人也惊呼起来，纷纷离座起身望向夜空。

　　蓝色的光芒逐渐变紫，逐渐逐渐地，在夜幕星河中组成一个巨大的沙漏，星光为沙，在时间内流转。那是丽莎的命之座。

　　“提瓦特的星空不会忘记每一个人，同时，她也会祝福每一个人。”我拿起两杯果汁，一杯放到了丽莎手里，“一年之中总有几个人的星座会在空中闪耀，而莫娜告诉我，你的命之座就在今日。所以，我约你出来可不是为了看流星雨，那样老套的套路，怎么会博得女孩子的欢心呢？”

　　丽莎没有说话，幸福地靠在我的怀里，眯起眼睛，沐浴着星光的祝福。

“我们的爱会像温妮莎树一样恒久，丽莎，”我附在她的耳边轻轻说道。“我以前行事从不许下承诺，只是模棱两可的回答。但对你，我会立下我生命中为数不多的誓言……”

　　“真的吗？”丽莎问道，舒展的眉头多出一抹阴云，她仍然不放心。“我见证了太多冒险家的离去，甚至还没来得及对他的亲人告别。如果你万一……”

　　“不会发生的，我不会让它发生的，丽莎。因为……”我说道，看着怀里柔媚的人儿，泪水从我眼中奔涌出来，我想要抑制，但却毫无作用。泪水模糊了她的容颜，我看到她金色的长发变成了缕缕耀眼的红丝，一双温柔的绿色眼睛变成了炽热的金瞳，透露着巾帼不让须眉的霸道:

　　“因为我是……被记忆诅咒的人啊……”

　　泪水流下脸颊，丽莎察觉到了气氛的异样。她伸出手拂去我脸上滚滚而下的泪珠，轻柔细腻的动作，让我看到被浓烟笼罩的天空，分不清是白天还是黑天。烈焰是唯一的光源，无数尸体堆在一起，走在上面如踏棉花，有种令人惶恐不安的柔软触感。我看到，火焰般的女子站在尸山血海之上，眼里闪烁着不屈的光芒，带着一丝诀别的不舍……

　　“还在想她吗？”丽莎说出了只有我们两个人能听懂的话。

　　“她是我一生的遗憾，丽莎，我们曾经立下海誓山盟，但……”我没说完，丽莎的唇便已经堵了上来。

　　“那么，你的第二次机会还被你搂在怀里呢～”

　　想去看到，想去听到，想要触碰到，可终究还是迎来了梦的结束。我看着那消失的星空，随着那景色一同模糊着，如梦般梦幻。丽莎，是缘分让我们相遇在乱世之外。

　　“星光无数，细砂无数，当有一星，发光予吾。”
