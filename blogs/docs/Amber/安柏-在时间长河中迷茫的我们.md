# 在时间长河中迷茫的我们

  今天蒙德城的夜晚没有像往常那样晴朗，月亮已经被暗紫色的云朵包围住。此时，一位穿着十分正式的男人敲开了安柏家的门。

  “你来了，契布曼先生。”

  “啊……啊！是……安柏骑士……”

  契布曼明显是有点紧张了。

  “别在那里站着啊，今天晚上的天气可不太好……”

  契布曼没说话，只是默默的坐在了那张为他准备好的桌子，桌子上的蜡烛散发着昏暗的光。透过那昏暗的烛光，可以隐约的看到一踏一踏厚厚的档案袋……

  “这应该是安柏您为明天演讲比赛准备的资料吧……”

  “嗯～并不全是……”安柏还是用那可爱的语气来回答。

  “哦~”契布曼的心放松了一点，看了看，安柏还是往常的骑士装扮，红衣服，红的长筒袜，白色的过膝长靴。安柏还是那么可爱啊！不过他不明白为什么安柏要在今天把他叫来。虽然自己在蒙德城中是一个小有影响的老板，但和安柏的交往并不多，为什么安柏会把他叫来呢？而且还在明天就该比赛的时候……还有，安柏她也要参加那场比赛。

  “对了，安柏骑士，你……额……是不是还要参加明天的演讲比赛啊？”

  “嗯，不过呢～我是替人参加的。”

  “替人？替的谁啊？”

  “杰瑞……”

  “啊？……这孩子啊……”

  契布曼的神色有一点紧张，因为这个叫杰瑞的孩子他听过，这孩子和他的家人在之前的一场案子中丧生。那个案子是震惊了整个蒙德城，因为伤亡的人数实在是太多了……

  “那个发生在福利社区的大案？”

  “嗯……”说到这安柏的神情有一些暗淡和失落，她把热好的咖啡摆在桌子上。随后，安柏也拿了一个椅子，坐在了契布曼的对面，从那一堆档案中拿出一个来，看了看，便放在了桌面上，叹了一口气……

  安柏：“那个……我们就步入正题吧！先从小杰瑞说起……”

   2月6日，那是一个宁静祥和的午后，蒙德城的人们都在午休，大街上的人少得可怜。蒙德城还是像往常一样平静呢。不过呢还是和平时有那么一点不同，就是到处都有施工项目。为什么会有这么多施工项目呢？那是因为在一个多月前，12月26日，琴团长提出了“蒙德七月改造工程计划”。意在用七个月的时间改善一下蒙德城的基础设施，补齐蒙德城的城建短板。“这么多的工程一块进行，我还真没见过呢……”安柏说道。

  “哎！小兔崽子！别跑！抓小偷啊！”

  安柏听到了以后立刻警觉了起来。“大太阳底下偷东西，也太大胆了吧……”安柏心里想着，已经做好逮捕犯人的准备了。眼睛扫了一下四周，隐约发现了一个身影。于是，安柏就追了过去。追了不到一百米就到了一个封闭的小巷子。安柏觉得这小偷也太业余了吧……

  “别……别拿弓箭，我……我不想死！”话语中透露着恐惧，声音是从一堆破木箱那传来的。而且安柏用她那双明亮的眼睛可以看出那堆箱子在发抖。看来这个人是真的怕安柏会杀了他……到了这，安柏有一点点疑惑和惊奇，安柏不知道那个人为什么会认为她会用弓箭杀死他。只见一个小男孩从箱子堆后探出头，眼睛里闪着泪光。蓝色的瞳仁在微微颤抖，是因为发自内心的恐惧。

  “孩子……”安柏弯下腰说着，她在尽可能的放低身姿的同时把弓和箭往身后放，双手放在孩子的肩膀上。

  “为什么会认为姐姐会用弓箭呢？”

  “因为几个星期前的那个……你用弓箭射伤的那个……呜呜”说着小男孩便啼哭起来。

  “几个星期前……”安柏正想着，后面传来一句话。

  “安柏！你抓住这个小偷啦！厉害啊！”

  安柏一听感觉声音怎么有一点儿耳熟，回头一看，原来是这街道附近的面包店老板，想必是他的店被偷了。眼看着面包店老板跑过来，小孩子立马躲到安柏的身后。相比起安柏，面包店老板还是更可怕一点。面包店老板跑过来定睛一看。

  “没错！千真万确！就是这小兔崽子！偷我家东西！安柏啊！你可是帮了我一个大忙嘞！”

  “真的是这个孩子吗？”

  “真的！千真万确！蒙德人不骗蒙德人！”面包店老板说到。

     安柏又看了看这个孩子，小男孩的衣服仔细看的话，应该是白色的，不过没有经常漂洗衣服上的灰尘和油污，白色的衣服已经变成让人难以直视的棕黄色。衣服的下面可以影影约约的看到一些伤痕。看到这里，安柏已经察觉到不对劲了。“其实……我觉得这孩子是不得已才去偷东西的……”安柏回答到。

  “嗐！哪那么多迫不得已啊！偷就是偷！要我说，这就一熊孩子！安柏啊！你就不要替他说话啦！”面包店老板愤愤的说到。

  “那个……”安柏欲言又止。

  “怎么啦，有话就说吗？”

  “那个……这个孩子给您的店面造成的损失……就由我来赔偿吧。”安柏有点不好意思的说到。

  “嗐！我还以为有什么事呢？安柏啊，你就直接把这臭小子给我，然后接着巡逻去。我啊！我就直接呱呱给这小兔崽子整俩嘴巴子，这事也就完了，那里用得着安柏你这么费心啊！”

  “那个……就当是我的一个请求可以吗？这个孩子给您的店面造成的损失，我都可以给您原价赔偿。如果您觉得不满意，我可以用更高的价格去赔偿您的损失。”

  “啊？嗐！既然安柏都这么说了。看在咱们安柏的份上，我就饶了这小子一马吧。”

  “那个……先生，您的面包店是在这附近吗？”

  “不是吧……安柏！你真的要替这孩子赔偿啊！”

  “就事论事嘛，原谅这个孩子，是因为我看这孩子实在是可怜，偷东西可能是迫不得已的，这是一方面。至于赔偿，这是另一方面的事。这孩子毕竟给您的店面造成了损失，我身为骑士肯定要为您解决损失。而且不能因为我觉得这孩子可怜就去忽视您的损失，这做无非是在慷他人之慨……”

  “哦，确实是这么个理，那么就跟我来吧。”面包店老板说到。

  不一会，安柏等人便来到面包店老的面包店里。

  “安柏，刚刚是我说话太过分了。其实啊，我面包店的损失没有那么大。这个孩子就偷了俩面包，而且这俩面包还是我们店里面最便宜的那种。”老板说到。

  “没事的，自己的店被偷，无论什么人都会生气的，你刚才的反应很正常的啦。”

  “嗐！安柏你说的这话太客气了！”面包店老板说着，便把脸扭到一边去了。他为自己刚才的冲动的话语而感到有些脸红。

  “孩子，你喜欢什么好吃的就尽管给姐姐说，姐姐都会给你买的。”安柏温柔的对小孩子说。小孩子一听到可以买面包，也不怕安柏了，把注意力几乎全部用在选面包上面了。

  “哎……小孩子就是小孩子，没心没肺的，一说到面包的就眼冒金光。”老板吐槽到。

  小孩子选了很长一段时间，拿了一大堆面包。

  “够了吗？”安柏轻轻的问到。

  “嗯！”小孩子高兴的回答到。

  “嚯！这么多，这小孩子可真是……安柏你确定都买吗？”老板问到

  “确定。”安柏回答到。

   “面包买好啦！姐姐现在要带你去西风教堂了。”

   “西风教堂？” 

   “对，西风教堂，那里有一位叫芭芭拉的牧师，很温柔体贴的。她可以给你身上的伤口进行治疗哦！”

   “啊……姐姐，原来你早注意到我身上的伤口了……”

   “嗯，孩子现在肯定很不舒服吧。”

  安柏和小孩子走在蒙德的马路上慢慢的走着，等到了西风教堂时，黄昏已至，太阳在西边留下一道暗暗的红色，夜空中已经有星星开始闪烁了。

  “唉……时间过得好快啊……对了孩子，你可以告诉我你叫什么名字吗？”

  “杰瑞。”

  “杰瑞？很可爱的名字啊！”

  “嘿嘿……”

  “杰瑞很喜欢面包吧，我看你拿了不少。”

  “其实，我吃一两个面包就够了，剩下的我想分给我的家人和同伴们……”

  “家人和同伴？”安柏疑惑的问到。

  “我的家人和伙伴，都在一个叫弗摩尔顿的工厂工作……” 

    “弗摩尔顿？”安柏对这名字太熟悉不过了，这个弗摩尔顿工坊，曾经叫恩格尔博工坊，是蒙德有名的工坊，产品闻名于提瓦特大陆，在蒙德的地位仅次于莱艮芬德家族的晨曦酒庄。可不知为何衰落了，后来又曝出了造假的丑闻，再后来就改名为弗摩尔顿工坊，虽然产品没有以前那么出名了，不过在蒙德仍然占有一席之地。

    “你的家人是那里的员工吗？我听说那里的员工待遇还是不错的。”

    “是不错，但这不是属于我们这些从出生到死亡一直呆在工坊黑暗处的‘榆木’。”

  “榆木？”安柏有些疑惑。

  “嗯，‘榆木’是我们的称号，负责管理我们的人叫‘毕夏普’。他对我们很不友好，他经常让手下对我们拳打脚踢，一旦我们不好好工作就抽打我们……”

    “抽打……”安柏听了小杰瑞的诉说后，赶紧拉上小杰瑞。安柏想要赶快找到芭芭拉，她觉得这件事绝对比自己之前的想象还要复杂。不一会，安柏她们便找到了芭芭拉。

    芭芭拉：“安柏？这个小孩子是……”

   安柏：“芭芭拉，这个孩子先托付到你这里，我去把琴团长叫来！”

   芭芭拉：“安柏！”

   安柏：“芭芭拉！剩下的事等我把琴团长她们叫过来再谈！” 



   琴和安柏赶过来以后，天已经完全暗了下来。

  安柏：“那个，请问一下，芭芭拉现在在哪里？”

  “就在那个位于走廊拐角处的房间，芭芭拉现在正在照护一个孩子呢。不过……我看到那个孩子满身的伤痕，也怪可怜的，不知道是谁这么狠心，竟然对一个孩子下这么重的手。”一名修女回答到。

  安柏：“好的，打扰了。”

  修女：“没关系，这是我们应该做的。”

  和修女打过招呼以后，安柏和琴就赶紧朝那个房间走去。

  琴：“安柏，假如我没猜错的话，那位修女口中‘满身伤痕的孩子’就是你说的杰瑞吧？”

  安柏：“嗯……感觉应该是他没错……”

  当、当、当。

  芭芭拉：“来了，这晚了有什么事情……呃呜！琴？”

  琴：“哦……我是听安柏说了杰瑞的事情才来的。对了，刚刚有修女说你在这里照护孩子。”

  芭芭拉：“没错……琴，我刚才正在这里给小杰瑞疗伤。”

  小杰瑞：“安柏姐姐来啦！”

  安柏、芭芭拉、琴：“嗯？”

  小杰瑞：“嘿嘿，安柏姐姐，芭芭拉姐姐真的好厉害，我现在身上感觉好多了！唉？那个个子高高的姐姐是谁啊？”

  安柏：“她是琴，西风骑士团的代理团长。你有什么事情都可以找琴团长解决哦！还有，杰瑞……如果你不介意的话……可以让琴团长看看你的伤口吗？”

  小杰瑞：“嗯……”小杰瑞很听话的将自己的后背露给琴看。

  琴：“嗯……看起来，这些痕迹应该是用棍子和皮鞭打的。”

  芭芭拉：“老实说，刚看到这些伤疤时我也吓了一跳。”

  琴：“孩子，你的经历安柏姐姐都给我大概说了一下，你是不是和一个大哥哥走散了？”

  小杰瑞听到了‘大哥哥’这一词后，情绪变得有些失落。

  小杰瑞：“他……死了……我躲在人群中看到他被一支箭射死在一个破棚子里了。”

  琴有些诧异。没错，就在几个星期前，1月2日，就有一名男性歹徒被射伤在一个棚子里。而射出这一箭的人，就是安柏。

  小杰瑞：“之后……我就一直在躲……几天前，我听到一些和我一样大的小朋友说是一位红衣服的姐姐射的那一箭。”

  小杰瑞：“所以我很害怕……我很害怕被那个红衣服的姐姐抓到，所以我刚见到安柏姐姐时会感到害怕。”

  说起来尴尬，琴已经猜到，小杰瑞所害怕的‘红衣服姐姐’就是现在站在他身旁的安柏。不过，这件事琴觉得还是先不要告诉小杰瑞了，毕竟现在最重要的就是让小杰瑞稳定好情绪，保证好小杰瑞的安全。

  琴：“孩子，不要怕。有我们在，那位红衣服的姐姐是不会找过来的。”

  小杰瑞：“真的吗？”

  琴：“真的。”

  芭芭拉：“小杰瑞一定很困了吧？让姐姐给你唱一首摇篮曲怎么样？唱完了，小杰瑞就好好的睡觉。”

  小杰瑞：“嗯！”

  “啦啦，啦啦啦～”伴随着芭芭拉温柔的歌声，小杰瑞缓缓的进入了梦乡。熟睡中的小杰瑞还时不时的抿一下嘴，整个房间都静悄悄的，只能听得见小杰瑞那舒缓的呼吸声。

  “看来小杰瑞已经睡着了。”芭芭拉悄悄的说。

  琴：“嗯，芭芭拉，你也赶快回去休息吧。今天让你忙到这么晚，也是蛮辛苦的。对了，我还有些事情需要和安柏谈一下。”

  芭芭拉：“嗯，琴，你和安柏谈完以后也要早点休息啊。”

  琴：“嗯，我会的。”

  说完，琴和安柏来到西风教堂的大厅，气氛上有些尴尬。双方沉默良久后，安柏开口说。

  “琴……我想去弗摩尔顿工坊调查一下。”

  琴：“嗯……说句实在话，这次我不能让你像上次‘黑火案’那样，独自一人前去调查了。”

  安柏：“……”

  琴：“安柏，不要太过于急躁。我们现在对弗摩尔顿工坊的情况了解甚少，连从哪里切入调查我们都还不知道。贸然调查肯定会打草惊蛇。身为代理团长，我不能让你去冒险。”

  安柏：“嗯……”

  琴：“安柏，1月2日的那件事还在困扰着你吗？”

  安柏：“没有，对于一个身负人命的人来说，自责是应该的……”

  琴：“安柏……这样吧，你明天先休息一天，在这一天里我和凯亚阿贝多他们商讨一下这件事。”

  安柏：“谢……谢了，琴团长。”

  和琴道完别以后，安柏回到了家里。桌子上摆着兔兔伯爵、一个照相机和一本书。照相机是枫丹生产的，而那本书是从须弥寄过来的。要说起这两件物品的来历，还是得和1月2日那件事说起。安柏躺在松软的床，缓缓的合上了眼睛，这几个星期的事情开始像走马灯一样，一幕一幕地慢慢的浮现在眼前。

  1月2日

  “别过来！再过来我一刀宰了他！”一个男人惊慌的喊着。

  琴团长：“咱们的骑士准备好了吗？”

  凯亚：“嗯，但我们的骑士并不能保证能击中那个人。如果要射击的话……很可能会让人质受伤……”

  “没办法，大团长远征时带走了不少经验丰富的骑士……”

  “安柏呢？”一名骑士问到。

  “安柏？”凯亚疑惑了一下。

  “安柏不光会飞行，而且蒙德所有的运动冠军都有被安柏包揽哦！”

  “哎，偏偏这时候把安柏给忘了！”凯亚说到。

  “快找到安柏！”

  “是！”这个骑士回答完毕以后便去找安柏了。

  “喂！安柏！”

  “嗯？”安柏回头一看，只见一名骑士向她跑过来。

  “出什么事了？”安柏问到。

  “有一个歹徒挟持了一名人质，躲的地方狠隐蔽，现有骑士的射箭能力都不行……而且更重要的是……这名歹徒情绪很激动！随时可能……嗐……抱歉刚刚跑的太猛了。”

  “嗯，我明白了，我现在就赶过去！”安柏说完这句话就立马动身了。安柏和这名骑士赶到现场时，看到了琴团长和凯亚正在部署人力。

  “琴团长！”

  “是安柏？”

  “是我，我听到了这名骑士给我的消息我就赶过来了！”

  “这可太好了，但是……”

  “有点不太乐观，歹徒现在已经不愿再看见有骑士增加了……”

  “不好，歹徒又有动作了。”一名骑士前来报告情况。“歹徒把人质带到一个小棚子里了，这下更不好观察了……”

  “看来只能试试从建筑物上找观察点了。”安柏回答道。“但这太危险了，又要隐蔽，又得注意建筑物的形状，稍有不慎……”琴团长说到。

  “放心吧，我可是侦察骑士，在这方面还是很在行的。”

  “行，一定要注意安全。”琴团长说道。

  说完安柏便如闪电一样离开了，一道红色的闪电在楼宇间稳健而隐蔽的穿梭。歹徒和那些与其对峙的骑士甚至都没察觉到有人在他们周围的建筑物间穿梭。安柏观察了一遍，经过反复的思索，最终确定一栋在歹徒躲避地方东侧的一栋房子。该房子的房顶上的杂物多便于隐藏，而且视野也不亚于其他地方。但这并不代表情况会有多么乐观。

  歹徒躲的小棚子的棚顶是一块破布，而且这块破布足以将小棚子掩盖的严严实实的。安柏只能通过破布上几块小小的缺口去观察。安柏将弓拉满，这次她不使用元素之力，对于歹徒只要制止其犯罪就可以了，杀死歹徒只是下策。安柏的目光集中在那几块可以观察歹徒的破洞上。经过几次的估算，安柏已经摸清楚歹徒的大致轮廓。他的重要部位在哪，怎么去射击，风力和风向对的影响箭道等……安伯的心中已经有数了。只见安柏深吸一口气，然后屏住呼吸，箭头在微微的调整，手臂慢慢的蓄力，弓缓缓的张开，手指轻轻的一放，箭便如一道闪电，唰！一声过后就听见一声惨叫。

  “啊！”歹徒痛苦的哀嚎道。他的肩膀在不断的向外喷血。安柏的那一箭正中歹徒的肩膀与右胳膊的连接处。疼痛、死亡、恐惧涌上歹徒的心头。他来不及管人质了，他仓皇的用那脏手去捂住正在喷血的伤口。被劫持的人乘机逃走，骑士们蜂拥而上，制服了歹徒，也掩盖住了这血腥的一幕。

  “制服住歹徒了！”骑士们兴奋的向琴团长喊到。

  “看来安柏做到了。”凯亚说到。

  “嗯。”

  “你的爷爷一定会为你感到骄傲的。”琴想到这不禁欣慰的笑了起来。不过，一个不幸的消息将刚松了一口气的琴团长拉回了紧张的状态。

  “不好了，琴团长，歹徒的身体状况不太乐观！”一名骑士说到。“赶快将犯人送到西风教堂救治！”琴团长命令到。

  “遵命！琴团长！”

  众人赶紧将歹徒搀扶到担架上，但此时歹徒的脸色已经有一点不行了。“不行，犯人现在危在旦夕，得在快点。”

  “我去前面开路！”安柏说到，在众人的努力下，用了不多一会的时间就到达西风教堂了，芭芭拉早在教堂里时就发现有人朝教堂这赶来，于是就赶紧出来准备进行救治。但就是在众人都这么努力的情况下，犯人还是没能坚持住。尽管芭芭拉用水元素修复了犯人的伤口，但依然改变不了犯人的虚弱情况。在经过几天的抢救之后，犯人还是离世了。

  收到了犯人的死讯后，琴立马带人来到了西风教堂。在观察了犯人的遗体后，琴思索了思索。

  “芭芭拉，犯人是因为什么特殊的病而死的？”

  “从病上讲没有什么特殊的，是死于普通的感染，但就我以往的经验来看，这位病人的免疫力应该比其他病人人要弱很多。一般人吃了药剂以后都会有很快的好转，但这个病人无论吃什么药都没有明显的疗效。对了琴，从这个人受伤到这里你们花了多长时间？我记得病人送来的面色口唇苍白，这可是重度失血的症状。”

  “大概不到半小时，而且我们还是给伤口做过止血处理。”

  “看来这位病人的凝血功能也有问题……”

  “芭芭拉，不如把犯人的遗体转运给砂糖吧，她正好是从事生物方面研究的。”

  “嗯，也好。”

  犯人的遗体封好后就被送到砂糖的研究室那里去了。把事情都安排好后，琴坐在椅子上，深深的呼了口气。这是琴为数不多可以放松的时刻，此时，又有一件心事泛了上来——如何安慰安柏。琴得知安柏自从得知了犯人的死讯后，就一直很自责，认为自己是亲手杀死犯人的凶手。

  “该怎么办呢？”琴自己小声地嘀咕着。

  “琴团长，刚刚有人往我们骑士团送了两个包裹，说是要送给安柏的。”

  “哦？”

  “那个，我们现在就去把包裹送给安柏骑士了。”

  “那个……安柏的包裹我来给他送吧。”

  “那也太麻烦琴团长了吧。”

  “没事，正好安柏最近有些心事，我可以趁送包裹的机会找她谈谈心。”

  “哦，那琴团长，我把包裹放这里了。”

  “嗯。”

  等送包裹的骑士走了之后，琴便走向那两个包裹并拿起来查看，只见这两个包裹上，一个上面的送件人写着“荧”，一个上面写着——“柯莱”。
