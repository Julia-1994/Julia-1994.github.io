# 这段感情就从这里开始吧？

　　“还有优菈。”诺艾尔的声音忽然弱了许多。

　　“旅行者？”听见这话，琴脸上也是闪过一片不悦。“找她干嘛？”

　　派蒙也跟着呼了一口气，“吓死我了！”

　　“咳咳！谁……说没事的？”荧哑着声音喊道。

　　“荣誉骑士前辈你是哪里不舒服吗？伤到哪儿了？能动吗？我现在就去找医师过来，你等会儿……”诺艾尔将荧摆好，马上就要往教堂那跑了。

　　荧忽然一把抓住她的手，将她拉住。“你等等！”

　　第二天一早
