# 名为诺艾尔的血色玫瑰-1

至冬宫

　　「博士，邪眼改良的最终实验如何了？」

　　「回禀女皇大人，邪眼的元素融合依然存在隐患，任何使用元素融合邪眼的普通实验体都会被能量反噬，至今已经实验了137次，均已失败告终，只有拥有神之眼才能使用邪眼的力量，而且不是共存，是等量替换」

　　「可有其他之法？两种元素尚且难以融合，何况六种，如果得不到所有的神之心，那元素邪眼要不惜一切代价成功」

　　「如果可以，我希望能得到一个实验样本，但是可能会比较麻烦」

　　「无妨，直言」

　　「女士曾提及的蒙德荣誉骑士空，可以在没有神之眼的情况下使用两种元素力量，他身上必然有异于常人之处，我想用他的身体做实验体定可以有新的进展」

　　「蒙德吗..这件事不能被西风骑士团发现，琴尚不为惧，但法尔伽...实验一定要不惜一切代价进行下去，叫五六来见我」

　　「遵命，女皇大人」

　　

　　蒙德

　　三个月后

　　“前辈，风花节这几天忙前忙后累坏了吧，今晚想吃些什么？我来做吧”

　　诺艾尔挽着空的胳膊，两人走在回家的路灯下

　　风花节这几天，两人都很忙，诺艾尔已有一阵子没见过空了

　　距离两人在一起已经过去了三年的时间，平稳的生活让两人渐渐的适应安宁，但空依然会在冒险家协会接受委托时通过各方渠道打听荧的消息，偶尔，诺艾尔也会看到空在寂静的深夜独自坐在椅子上看向窗外，脸上漫溢伤感的神色，但即使诺艾尔有所察觉，她的心里也有着说不出的无力，她不知道该怎么帮空，不知道该如何替空分担

　　“我的诺诺这几天不也是为了大家东跑西跑，看看，都瘦了，不要做了，去猎鹿人买一点带回去吧”

　　空心疼的伸出手捏了捏诺艾尔的脸，与前几日相比，确实肉眼可见的消瘦了许多

　　两人去猎鹿人买了些吃的，便回了家

　　“前辈，我去厨房拿碗筷，你先去换衣服吧”

　　“嗯，好”

　　说着，空便推开卧室的门，一阵凉风袭来，让空绷紧了身体，他分明记得早晨走的时候已经锁死窗子了，为什么回来的时候却是开着的，空握紧剑柄，警惕性打量着四周，却突然看到一封蓝色的信躺在桌子上，缓步向前，拿起信封拆开

　　「若不想荧死，三日后亥时，一个人前往孤云阁南岛」

　　只有短短一句话，但却字字关键

　　「果然，该来的还是要来了吗」

　　空心里默默的想着，透过门缝可以依稀看见正在厨房收拾着的诺艾尔，这三年他一直没有放弃寻找荧，但从来不会在面前诺艾尔提及此事，更是从未向外人提及此事，只是默默的承受着这份压力，所以无论信是真是假，他都要去确认，即使字里行间充满了危险，即使知道有可能有去无回

　　“喵”

　　这是两人去年在猫尾酒馆领养的猫咪小艾，它过来蹭了蹭空，让空回过神来，找了一本书将信塞到里面，把书放回书架后，便把猫抱起来

　　“走吧小艾，看看妈妈收拾完了没有”

　　空抱着小艾坐在沙发上看着还在忙的诺艾尔

　　“诺诺，之后两天请个假吧，风花节要结束了，没什么太多事情可做，我们也趁这个机会放松一下”

　　“前辈想要去哪里玩呢？明明还有些收尾工作没有完成”

　　“想去挺多地方的，但是都想和你一起”

　　“好啦，我明天去找琴团长请个假啦”

　　看着把小艾举在头顶可怜巴巴盯着她的空，诺艾尔无奈的答应下来

　　饭后，卧室里，诺艾尔躺在空的腿上抱着小艾看书，随口问空

　　“我们明天要先去哪里？”

　　“嗯...誓言岬，摘星崖，风起地这些地方都想再去看看吧”

　　“前辈真是的，这些地方明明经常可以去看到嘛，为什么要挑这两天呢？”

　　空沉默着低头看正在一边给小艾顺毛一边看书的诺艾尔，在家的时候诺艾尔总是很粘着他，他伸出手抚摸着诺艾尔整齐的头发，平静的生活终归是被打破了，这种看似轻松的日常在自己离开后不知道还能不能再拥有，这三年，空对腿上躺着的这个乖巧懂事的女孩万般的珍爱呵护，生怕诺艾尔也会像荧一样有一天离开自己，但即使想到有一天自己会离开，却没想到这一天到来的这么突然

　　“两天...吗”

　　空轻轻说着

　　第二天清晨，空意外的比诺艾尔醒的都要早，旁边的诺艾尔抱着他的胳膊脑袋靠在肩边还在熟睡，空靠近她的额头轻轻亲了一下，随后给她掖了掖被子，便起身去厨房做早餐了

　　一个小时后

　　“起得好早啊前辈”

　　看着打着哈欠歪着脑袋眼睛还没彻底睁开的诺艾尔走出来，头发和睡衣都乱乱的，空宠溺一笑，拿起旁边的梳子招手让她坐在自己的腿上，帮她梳理着头发

　　“这两天太累了吧，早饭还要一会，再睡一会吧，时间还早，天都还没亮呢”

　　“陪我一起睡吧，看到你不在我就出来了，我自己睡不安稳，总感觉没有前辈空空的”

　　“好，那先去卧室吧”

　　“嗯...”

　　话还没说完，诺艾尔已经靠在空的怀里睡着了，脑袋抵在空的颈窝，轻轻放下手中的梳子，拿起旁边的毯子给诺艾尔和自己盖上，就这么抱着她，也渐渐睡了过去

　　等到两人再次醒来太阳已经升高，一起吃过早饭，洗了澡换好衣服便出了蒙德

　　牵着诺艾尔的手走过风起地，和煦的阳光铺洒在两人身上，三年前两人在这里确认了心意，诺艾尔闭着眼靠着空，风轻拂她的发间，吹向树叶，沙沙的响声伴随着两人的前进，这里有很多风花节期间为了表达爱意的情侣在这里献上的花束，风车菊，塞西莉亚花，甜甜花...各种各样摆满了树根

　　两人顺着之前的路走着，走到了鹰翔海滩，以前的夏天，诺艾尔总是和空来这里吹着海风散步，听着小孩子抓螃蟹的欢声笑语，聆听着海浪敲打岩石的声音，和空说着或许海的另一边是另一个新的世界，那里有成片的花圃，里面种满了玫瑰

　　到了摘星崖，星星已经渐渐布满天空，两人拿出背包里的帐篷，很快就搭建了起来，吃了些随身带的食物，坐在一起看着美不胜收的夜空

　　“前辈，是发生什么事情了吗，总感觉你今天话很少”

　　“诺诺，你陪葛罗莉小姐在这里放飞蒲公英的时候，有说过是为什么吗？”

　　空还没有回答诺艾尔，反而问向她

　　“葛罗莉小姐说，蒲公英可以给想见却又见不到的思念之人带去好运”

　　“这样吗”

　　空点了点头，拿摘下一朵蒲公英，微弱的风元素带起它的种子，空凝望着它渐渐飘向远处，诺艾尔没有追问，只是默默陪着空摘起一朵蒲公英和他一起放飞，她不知道发生了什么，也不想逼问空，她知道，能让空这样的，除了她就只有荧了

　　没过多久，两人便在篝火旁相拥而眠

　　第二天，两人又去了果酒湖，看了风龙废墟，最后来到了誓言岬，在这里，依然看不到海的尽头，或许海根本就没有尽头，誓言岬的一位老人曾在这里留下石碑「人对于事物过于期待，只会带来更大的失望，有时候没有希望与期待或许会更好」但老人并不知道，真正坚定的人并不是为了坚持而去获得什么，而是因为有了期望之物所以选择坚持

　　等到两人游遍了想去的地方回到家，天色已经很晚了，空累的趴在床上不想动弹了，等到诺艾尔收拾好喂了小艾换完衣服回来，空已经睡着了，拍醒空帮他脱下衣服和鞋子，诺艾尔也靠在空怀里沉沉的睡了过去

　　凌晨，空睁开眼睛，眼神清明，轻轻挣脱像往常一样抱着他胳膊的诺艾尔，起身穿好衣服，坐在床边，借着月光抚摸着诺艾尔的脸颊，眼角丝丝泪水凝聚，低头在她唇间一吻，他知道，再这样下去，他就舍不得，走不了了，拿出一封早就准备好的信留在桌子上，心里默默的想着，「再见了，诺诺」，在小艾的注视下离开了家，离开了蒙德

　　清晨，诺艾尔被小艾的蹭醒，睁开眼，看到小艾趴在她身上的被子，旁边空已经不在了，甚至一丝丝温度都没有，一种不好的感觉涌上她的心头，她起身走出卧室，厨房也没有见空的身影，冷清的家里只剩下她自己，小艾跟出来咬着她睡衣的裤脚，把她拉进了卧室，坐在信上，诺艾尔颤抖着伸出手，打开了信

　　「诺诺，你看到这封信的时候，我已经离开蒙德了，我不知道还能不能再回来见到你，也不知道还能不能再陪你去风起地和摘星崖，原谅我的不辞而别，总有一些事要我去完成，这是我的责任，我也不能把你带入我所要负责任的危险中，在这个世界，我本无亲无故，但这三年，与你生活中的点滴让我缓解了失去亲人的伤痛，你不仅是我的爱人，也是我生活的一部分，如果我能活着回来，就在蒙德大家的见证下与你举办婚礼，如果我没有回来，你也要好好活着，不要做傻事，小艾还需要人照顾，我走后，希望那天的蒲公英能给你带来好运———永远爱你的空」

　　诺艾尔强忍双眼把持着泪水看完，脑海已经一片混沌，原来那朵蒲公英是为自己放飞的吗，明明不想去想空回不来该怎么办，可偏偏控制不住自己，控制不住自己的想法和心，身体微微颤抖，终是双眼把持不住泪水，一串串的泪珠从好看的双眸滚落，浸湿了睫毛，双手将信抱在怀里蹲在地上抽噎着，她不想空丢下她离去，哪怕是死，她也想和空死在一起，但她并不知道空去了哪里，也没办法找到他

　　小艾跳到诺艾尔的腿上舔了舔她的泪水，咬着她的衣袖，拉着她的手向着书架的方向，那里的一本书中，引起一切事端的蓝色信纸还在里面安静的夹着
