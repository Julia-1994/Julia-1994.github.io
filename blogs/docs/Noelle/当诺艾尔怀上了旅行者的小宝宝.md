# 原神同人文：当诺艾尔怀上了旅行者的小宝宝（桃文）

统计信息：字数7441  阅读15分钟  短篇小说


 “诺艾尔！”

  蒙德的某个角落又传来了呼唤声，若是在平时诺艾尔肯定会立刻来到他的身边，但是这次却过了半晌都没有动静。

  或许是没听见，优律这样想着，于是她又大喊了一声。

  “诺艾尔！”

  但是依旧是没有回应，过了三四分钟之后才看到诺艾尔姗姗走过来，样子有些奇怪，红着个脸步伐也十分小心，就像是怕受伤一样。

  “诺艾尔啊，帮帮我，我现在有个麻烦。”看到她走过来优律小姐自然说出了请求，但是未等她说完，诺艾尔有些不好意思的说道。

  “那个，优律小姐很抱歉，我恐怕不能帮你。”

  “诶？怎么了？”优律有些意外，平时不管说什么诺艾尔都会欣然答应下来，但是现在却拒绝了。“是有其他事情么？”

  “算是吧...”诺艾尔小声说道，没有得到帮助优律并不觉得生气，她平时受了诺艾尔那么多的帮助，现在是时候关心一下她了。

  是生病了么？优律心中的第一个想法便是这个，随即她伸手去抚摸诺艾尔的额头，但是温度确实正常的，看样子也不像是生病，还是说有其他的难言之隐？

  “诺艾尔你怎么了？肚子不舒服？还是说最近身体疲劳了？”优律关心的问道。“多休息一下吧，好好休息才是重要的。”

  “谢谢你优律小姐，我只是...”诺艾尔很明白自己怎么了，但是却有些耻于说出口。

  这时候优律想到了一些其他的东西，身体有些不舒服并非是发烧一类的，而且看起来很难开口，难道是...

  “诺艾尔，你好好休息吧，我知道你怎么了。”优律笑了笑，随后拍了她一下肩膀。“嘛，女孩子总有那种时候，亲戚来了就不要勉强自己了。”

  诺艾尔楞了一下，等到反应过来的时候她赶忙附和道。

  “对对，是的，是这样的，真是让优律小姐担心了。”

  “没关系，我们都是女人，都懂的嘛。”优律笑了笑，随后继续说道。“需要我送你回去么？还是说....”

  优律看向了不远处，旅行者现在这个时间就在图书馆呆着，整个蒙德的人几乎都知道旅行者和诺艾尔之间的恋情，这根本不是什么新鲜事情。

  “要荣誉骑士送你回去？”

  “我，我自己就可以了...”诺艾尔笑了笑，这时候旅行者也走了过来，看到旅行者，优律故意摆出了一副严肃的表情，像是在斥责。

  “荣誉骑士，女朋友身体不舒服你可是要照顾好她得，怎么自己去别的地方了？”

  “我下次会注意的，去买了一些水果，我和诺艾尔这就回去。”旅行者明显是有些慌张，像是怕被察觉到什么一样。说着她便拽着诺艾尔的手，离开了这里。

  他们手牵着手走在蒙德的街道上，离家的路途并不远，没过几分钟便到了家门口。

  “诺艾尔，我都说了你不要出来了嘛，这件事前别让别人知道。”旅行者有些报怨似的说道，随即他掏出了钥匙打开了门。

  在开门的一瞬间一股花香扑鼻而来，不管多少次旅行者都觉得这香气是那么的舒爽，房间也被诺艾尔打扫的一尘不染，就像是新家一样。

  餐桌上自然是摆满了点心，这些都是诺艾尔做的，她总是愿意做一些小点心来迎接劳累了一天的旅行者，虽然某种意义上她比旅行者还有辛苦。

  “这段时间就我照顾你了，你不要乱走动了。”

  “可是大家要找我帮忙的话，我不去会觉得罪恶的。”诺艾尔低下头不好意思的说道，旅行者叹了口气，她似乎一点都没有变，永远是蒙德城里最任劳任怨的那个人。

  若是平常的时候，旅行者可能就跟着诺艾尔一起帮助别人，可是现在不一样，诺艾尔的情况十分的复杂，要是按照普遍理性来解释的话，就是怀上了自己的小宝宝。

  这样的事情让别人知道了的话，旅行者肯定不会有什么好下场，毕竟诺艾尔才未成年，琴知道大概会打死自己。所以他才让诺艾尔这么小心，在事情没有圆满解决之前尽量少出房间，像是家务和做饭什么的都交给自己，这段时间她需要好好休息，而不是去帮助其他人。毕竟诺艾尔是个普通人，要是现在还想着去热心肠，那就真是诺艾尔屹立于大地之上了。

  “那么就帮我一个忙，这段时间你好好呆着，并不是谁为女仆，而是身为我的另一半。”

  听到这话，诺艾尔的脸颊不禁泛起了一阵红晕。

  “好，好的。”她笑着说道。但是就在这时候....

  “诺艾尔！”

  这声音吸引了旅行者向那个方向看去，但是等到回过头的时候...

  “诺艾尔？诶？人呢！？”

  诺艾尔早已经不见，只留下一个被打开的窗户。而此时的另一边，丽莎的图书馆里。各式各样的书散落了一地，因为丽莎的一些失误现在让这里变得乱糟糟的。

  “啊，你来了，诺艾尔帮我收拾一下书籍吧。”

  这样的小事诺艾尔还是可以做到的，她随即答应了下来，蹲下身子捡起了书籍。而大概五分钟后旅行者才找到诺艾尔，他气喘吁吁的扶着门框，然后对还在收拾书籍的诺艾尔说道。

  “诺艾尔，我来，你回去休息吧。”

  “啊呀，是旅行者啊。”丽莎看见旅行者走了过来笑了笑，眼神中带着某种色气，当然旅行者不会被这吸引，他心里只有诺艾尔。“我以为你已经被诺艾尔养成废人了，没想到现在还会干杂活。”

  “才不是。”旅行者大声纠正道。“我一直很自立的好吧，诺艾尔身体不舒服，我来接手了。”

  “诺艾尔身体会有问题？去雪山救人了？”丽莎有些惊讶的说道，在她的印象里诺艾尔也只有这种情况会生病。

  “这，原因很难和你说，总之我接手就是了。”

  看着旅行者的样子貌似有些难为情，而看向身边的诺艾尔的时候，竟然发现她也红着脸。突然之间丽莎有了一种预感，手中的法术图案显现了出来，然后对着诺艾尔的肚子扫描了一番。

  但是之后的结果并不像她预想的那样，她有些失望的说。

  “切，我以为怀孕了呢？”

  听到这话两人同时看向了丽莎，旅行者更是有些兴奋的说道。

  “什么？诺艾尔没有怀孕？”

  “我的魔法不会出问题，根本没有怀孕，我一开始还以为...等等，你好像知道？”

  “额...是这样吧。”旅行者傻笑了起来，而诺艾尔也松了一口气。

  “吓死了，我还以为以后不能帮助大家了。”

  所以这时候的情况算是皆大欢喜，旅行者也意识到了自己之后要加强安全措施，可是他没有注意到的是，琴团长正站在他的身后。

  “旅行者，你把诺艾尔怎么了？”

  听到这个声音，旅行者突然认识到，他可能要完蛋了。

  过了几分钟后，禁闭室内。

  “啊，旅行者你好啊，是来找可莉玩的么。”

  “让我静一静。”
