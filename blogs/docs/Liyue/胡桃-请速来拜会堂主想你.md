# 请速来拜会，堂主想你

统计信息：字数4076  阅读9分钟  短篇小说


请速来拜会，堂主想你！

璃月港。

夜，灯火熹微。

万籁俱寂。

进了璃月港，同夜间值守的千岩军核对过身份，荧便一身风尘的匆匆赶往往生堂，脸上满是掩饰不住的疲惫。

早在一个月前，还在望舒客栈里她就收到了胡桃的信函。信上邀请她整月之后前来往生堂一聚。

只是当时荧未能得空，一直在处理早先盗宝团遗留下的麻烦事，一直到半个月前才勉强将事情做完。

事情一完成，甚至来不及多休息，荧便日夜兼程的往璃月港赶来，生怕错过同胡桃约定好的日子。

就连派蒙一路上絮絮叨叨的抱怨她都没多说些什么。

荧沉默着走着，派蒙这会儿趴在荧的头顶呼呼大睡着，打着细微的鼾声。

这些天连夜赶路，别说荧，就是一直摸鱼在半空飘的派蒙也都累坏了。

往生堂近了，一道人影站立在门外，在暗淡灯光的照耀下，看上去有些孤独，形单影只。

见到那道身影，荧心头不由的一颤，加快了脚步，一溜小跑的赶到往生堂前。

那道人影仰起头，赫然正是胡桃。

仰起头，见到荧这一身风尘仆仆的模样，胡桃心中不由的也是一颤，两步上前，把住荧的臂膀，抿了抿唇有些心疼的道：“荧，对不起，我……”

荧忙伸手捂住了胡桃的嘴唇，笑了笑，道：“没事，堂主相约我怎能迟到，我们先进去吧。”

胡桃点了点头，二人把臂走入往生堂。

过了些时候，将派蒙在一楼的客房安置好，荧洗漱干净换上一身宽松的睡衣，缓步走上二楼胡桃的房间。

推开门便看见胡桃端坐在床边，身体有些紧绷，看上去很是紧张。

荧上前坐到堂主的身旁，伸出手抓住胡桃的手掌，五指交缠五指相扣，轻笑道：“抱歉，让堂主等我这么久，这么些时间，想来堂主一人也没那么好过吧。”

胡桃稍稍用力，抓紧了与荧五指相扣的那只手，略略仰起头，在灯光迷蒙下看着荧的面孔，眼睛不自觉就眯了起来，弯成了月牙的轮廓。

胡桃低声细语，柔柔地道：“你不在这些日子，可把我孤单坏了，你要是再不回来，我都不知该怎么办才好。”

荧笑了笑，眼睛也不自觉的看向胡桃，却见胡桃披散着头发，脸蛋红扑扑的笑着、笑着，眉眼盈盈，如落一潭春水，煞是动人。

荧稍稍靠近了胡桃些许，直到两人肩抵着肩、臀抵着臀的地步，而后伸出另一只手熟练的撩起胡桃上衣的下摆，手掌轻轻贴住了堂主微微鼓起的温热小腹。

“唔——”

胡桃不可自抑的低低哼了一声，夹紧了一双白白嫩嫩的大腿，脸上又扬起几分可疑的红晕。

胡桃低声说道：“荧，你轻点，憋了不少，我可禁不起你的胡乱折腾。”

荧忽然松开握着胡桃的手，一把抓住她的肩膀，略一用力便在胡桃的惊呼声中把胡桃放翻在床上。

荧趴在胡桃的身上，用自己的小肚子抵着胡桃的小腹，轻轻蹭了蹭，扬起眉毛笑意盈盈地说道：“堂主大人，同样憋了不少的又不止你一个。”

在胡桃讶异的目光中，荧扬起身子跪坐在胡桃的身上，一点一点解开了自己上衣的扣子。

衣衫大开，那显露出的同样鼓起的小肚子瞬间就吸引了胡桃的注意力。

胡桃抬起手，用一种向上托举的姿态轻轻抚着荧的下腹，感受到其内蕴藏的重量，也笑了起来：“这下子我们一样了呢，荧。”

胡桃低低唤着荧的名字，拉着荧的手便也将她重新拉倒在自己的身上，而后伸出手环过荧的后颈，轻轻地搂着她，面颊贴在一起，耳鬓厮磨。

夜晚沉入海底，距离黎明的破晓尚有一段不短的距离。

这个夜晚不为人知的温存也将成为旅者与胡堂主之间的下一个小秘密。

夜深人寂时，暖风微醺，白月皎皎。

正当时。
