# 椰羊的正确打开方式》（刻晴/甘雨）

  天衡山与璃月港相邻，相比富饶繁荣的璃月港，按理说这里怎么也至少应该是一个3A级旅游景点一般的开发水平。然而，不知道是仙人的恶趣味、还是璃月人民的疏忽，这座山上除了零零星星的几座小居之外，俨然是一幅未开发的荒山模样。

  等等，修厕所这种事情，明明应该是璃月七星做的……
