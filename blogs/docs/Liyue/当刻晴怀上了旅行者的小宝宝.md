# 当刻晴怀上了旅行者的小宝宝（桃文）

统计信息：字数6852  阅读14分钟  短篇小说


不论工作到何时眼前的工作总像是永远做不完一样，虽说可以留给明日再做，但是刻晴总感觉明天会有更多的事情。

最近几天总是有些头昏脑涨，吃饭也有些吃不下，就像是生病了一样....倒是一旁在帮忙的旅行者闲的很，嘴里还哼着什么歌。

“Just Fight Cosmos~”

这还在唱，就差喊出来了。这是哪里的语言刻晴都没有听出来，或许是蒙德语？

要不还是休息一下吧，刻晴扶着自己的额头想到，自己的身体这几天每况愈下，不会真的是得了什么病吧。

“怎么了？老婆。”这时候旅行者看到刻晴的样子有些不舒服，走过来关心道。

刻晴听后立马紧张了一下，她赶紧看周围有没有人，在看到周围并没有其他人后，才略带嗔怒的说道。

“你说那么大声干什么？万一有人怎么办。”

“没事的，她们不是都知道么。”旅行者倒是满不在乎，毕竟玉衡星和一位异乡来的男子有很亲密的关系这件事情早已在璃月传开了。

“那，那也不行这么叫。”虽说如此，刻晴还是羞红了脸。“外人面前，要叫玉衡大人。”

“好好好，玉衡大人。”旅行者十分顺从的说道，还摆出了一个请安的姿势。“不知玉衡大人是否要回去歇息了？”

虽说时间还早，但是今天却是是有些累了，先回去吃饭然后工作就拜托甘雨先处理一下吧...

随即刻晴便站起了身，而旅行者也跟在她身后一起走出了办公室。

这时候正好是晚饭的时间，因此旅行者问刻晴道。

“玉衡大人，今晚吃金丝虾球？”

“不了？”刻晴挥了挥手，随后扶住了额头，这几天她有些厌食，也不知是为何。“我想吃一点酸的东西，麻烦你帮我买回来一些酸蜜饯吧，我记得码头那边有家店。”

“刻晴小姐，您是不是生病了啊？”这时候派蒙也在一边说道，这个小家伙明显是看出来刻晴的不适已经到达了一定程度。

“也许吧...”刻晴无力的说道。“麻烦你去买些水果和蜜饯什么的，我先回去休息了....”

“好的。”自己最喜欢的人下的命令，旅行者自然是不敢怠慢，立马跑向了码头。而刻晴也快步回到了自己的住所，浑身无力的坐在了位子上。

“好难受啊...”刻晴嘟囔着，此时她赶紧自己浑身没有力量而且非常的想睡，但是她还是强忍着困意坐了起来，走到镜子面前看了看自己的样子。

若是生病的话皮肤应该变得惨白才对，可是气色很正常啊。

刻晴认真的想着，自己这样懈怠的工作太多长久下去不是个办法，必须解决才是，要不和旅行者说一下最近几天好好休息？

就在她思考的时候，旅行者也刚买完她要的酸的水果和蜜饯，走进了房间。

“老婆！东西买回来了！”

“嗯，现在放在桌子上吧。”离开了办公室倒是没有那么多顾忌了，旅行者叫她老婆也没有那么大的反应。

虽说刻晴说了不要金丝虾球，但是旅行者还是买了一些，他也知道最近刻晴最近吃的很少，就像是换了一个人。

随即旅行者将虾球，水果，蜜饯什么的一一摆上了桌子。此时也正巧是玩饭时间，两人坐在一起，开始享用晚餐。

刻晴期间一直不说话，在思考着自己究竟是得了什么病。而旅行者也是一直在说最近工作的情况和见闻什么的，说个不停。旅行者其实也有些奇怪，自己游历了这么多世界，也只有刻晴吸引了自己...那么在她身上付出多一点关心也是没错的吧...

或许他此刻短暂性的忘记了，自己还有个妹妹没有找到。

“对了，那个...”刻晴犹犹豫豫的说道。“我们这几天休息一下吧...身体有些不舒服。”

虽说刻晴还是蛮喜欢和旅行者在一起时候的那种感觉，但是工作最重要，最终她决定还是休息几天。

旅行者当然也是理解她，毕竟身体才是第一位。

“那这几天就好好休息吧，要不要明天看一下医生？”

“嗯。”刻晴点了点头，然后拿起了一枚山楂。

“来吃一个虾球吧。”旅行者夹起一块虾球，停在了她的嘴边。“来~啊~”

这种暧昧的动作若是换做一对刚相处的情侣或许会脸红起来。但是二人相处了这么久，一切早已习以为常。

刻晴十分自然的吃下了那颗虾球，然后又拿起一个水果。稍稍吃了几口，便感觉自己已经饱了。

“回去睡觉吧，最近好累啊。”刻晴不禁又报怨了一句，然后站起身来走进了卧室。

她躺在床上，闭上眼睛便又想到了最近的一些工作，郊外的土木工程开发和一些建筑翻修的事情，不仅让她头痛不少。

还是要快点将身体调整回来，这样才能继续好好工作...而这时旅行者也一下子扑了过来，将被子盖在两人身上。

但是却在此时，旅行者却搞起了小动作...

“你干什么啊...”

“难免有些无聊嘛老婆~”

“那你不能，等等，你干什么？”

“最后一次，最后一次好不好？”

“.....最后一次哦·~然后我好好休息几天。”

第二天早上

“诶~医生我怎么样了？”

经过昨晚的折腾，刻晴不免觉得自己的身体更虚了一些。

医生在仔细的思考着，而旅行者也在一旁等待着结果。在漫长的等待后，医生终于缓缓开口道。

“不知玉衡大人您，最近几天是否厌食，无故干呕，嗜睡，四肢无力，喜食酸甜？”

这些都说到了刻晴这几天的状态上，无需纠正些什么，刻晴自然的点了点头。

医生有思考了一阵，然后又测了一下刻晴的脉搏，反复确认直到拿下结果后，医生站起身来，做出一个恭喜的手势。

“恭喜玉衡大人，贺喜玉衡大人。”

“诶？这是做什么？”刻晴被这突然而然的道贺搞得有些疑惑。

“玉衡大人，您有喜了。”

“啊！？”刻晴和旅行者还有派蒙几乎是同时发出了惊讶的叫声，刻晴不敢相信自己竟然有了....而旅行者也不敢相信刻晴有了自己的孩子。

“这，这...”刻晴结结巴巴着不知说什么好，她随后看向了身旁的旅行者，气不打一处来，一脚踹了过去。“都怪你！没事晚上折腾的那么勤干什么啊！”

“老婆，息怒！孩子！息怒啊！”
