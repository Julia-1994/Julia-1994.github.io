# 原神同人文：丽莎的惩罚（桃文）

统计信息：字数4929  阅读10分钟  短篇小说


（简单介绍一下，主角：空，没有派蒙，属于平行宇宙，后宫向，第一章可能有点问题，第二章再改）

  解决风魔龙事件之后，已经过去半年了，在这半年里，我早就与蒙德城的各位混熟了。

  今天在凯瑟琳那接了一个任务，任务要求我去骑士团去面见凯亚，他会详细的说明，任务信息，当我赶到骑士团时，凯亚坐在椅子上，右手抚摸着下巴，左手拿着一封信。

  看见我进来后，没等我开口，他便插在我前面说：“哟，空，这次是你接的任务啊，很好，如果是你的话就很好办了，你先把这封信送到丽莎那，就图书馆里，很快就上来了。”

  我警惕的看着他“别，你先说是什么任务，我再去帮你送信，不然我不会去的” 凯亚沉默了一下，随机露出了一个笑容:“哈哈哈，空，你还是这么有警惕性啊，其实这一次的任务就是帮我送信，这里不仅有丽莎的，还有其他人的，我腿上一次跟着琴团长去剿灭坠星之地七天神像周围的怪物时，那一次，我看到了令我最恐怖的东西，突然他顿了一下，看到我这么全神贯注的听故事，他突然不说话了，吊着我的胃口。

  我不满的看着他“然后呢？然后呢？你到底看到了什么？”我开始猜测下一步的走向，他露出了一个笑容，这个笑容使我胆战心惊，“这样，这里有五封信，你先去送完信之后，我再慢慢的跟你讲我那天在七天神像那，看到了什么”

 没办法的我 只好乖乖去送信，大约30分钟后，我跑遍了整个蒙德，就只差丽莎的信没送达，因为我找不到她，所以只好先留着，等明天找到她才能给她。

  我跑到凯亚的办公室，火急火燎的说“凯亚！全部送完了，现在跟我讲你到底看到了什么！” 凯亚愣了一下，笑了笑，喝了口水继续说“那天我看到了……”他故意拖着长音，“什么，究竟是什么！”我已经等不及了，“我看到了一只兔子，真是太恐怖了”。

  “兔…兔子？”我用不可置信的语气问着他“是啊，太恐怖了，看到兔子我总想忍不住的去狠狠揉他脑袋，于是，我走过去的时候，踩到木棍，把腿摔伤了”说完他又喝了一口水，眯着眼睛看着我

 我沉默着转身，准备离去，“要是丽莎的信没送到，那就遭了” 凯亚的声音还在后面说到，但我已经听不进去了，沉默的走回去，脑子里全部都是，兔子 原来是兔子，为什么会是兔子等一系列想法，浑浑噩噩的走到家，喝完床头的水，准备以睡觉抚平我内心的伤痕。

  一种强大的疲惫感向我袭来，在失去意识之前，我仿佛看到了一个身穿紫色连衣裙的人，在那笑着。

  当我再次醒来时，发现我在一处昏暗的屋子里，手脚被绑在床上，浑身使不上力气，我原以为是雷莹术士他们寻仇把我绑架的时候，一阵熟悉的笑声传来，我艰难的转过头看，发现是个老熟人，丽莎

  “吓死我了，丽莎姐姐，你这是干嘛啊，我没有借书不还啊，我也没有损坏书籍，放我出去”我在那向丽莎提出反抗，回应我的只有丽莎的笑声

  “小可爱❤️～我可是听说，你把姐姐的信偷偷留着，你想干什么呢～”

  丽莎走到床边，用手轻轻抚摸我的脸庞

  “丽莎姐姐，你听我解释！我找不到你，只好把信放在我这，准备明天拿给你的！你要相信我”

 我眼神里充满恐惧，充满对未知的恐惧 

  “哦呀～似乎是错怪你呢了小可爱❤️～但是这封信，五天前就该在我手里的呢～但你拖到了现在呢～”

  丽莎的手开始从我脸颊上慢慢往下移，最后停留在我的胸膛那，我咽了咽口水，小心翼翼的说

 “丽莎姐姐，要不你放了我怎么样，我一定给你赔礼道歉，能不能先别在我胸膛那画圈了” 

  丽莎的动作停了，似乎正在思考什么，随即微笑着说着“哦呀～小可爱不喜欢这样吗？真是伤脑筋啊，这样呢❤️～” 丽莎把手放到了最下面（懂得都懂）

   “逃不掉吗？” 

    “嗯～” 
    
    “我还有其他选择吗？”
    
    “没有哦～亲爱的❤️～”

   我深吸一口气，喊到“凯亚！你算计我！” 

   随即向丽莎说到

 “丽莎姐姐，轻点～我怕疼～” 

  丽莎，顿了顿，笑着说

   “介于，你刚刚喊了别人的名字，所以有小惩罚哦❤️～”
